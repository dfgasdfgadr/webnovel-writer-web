# 第一卷：第六面 - 节拍表

> 章节范围: 第1-75章
> 核心冲突: 封不觉发现诊室的秘密，完成第一批执念案件，搭档身份初步揭露，组织第一次逼近
> 卷末高潮: 诊室被组织发现，搭档为保护封不觉暴露自己行踪，卷末记忆回流第一块：封不觉回想起七岁时目睹的第一个死者

## 1) 开卷承诺（Promise）
- 本卷读者承诺（爽点/悬念/情绪）：
  1. 每案一次执念完成的释放感——死者不再是无名的鬼，而是有故事的人
  2. 搭档身份的层层揭开——一个"已死之人"为什么活着、在躲谁、笔记本上写了什么
  3. 组织逼近的压迫感——每一次正确选择都让敌人更近一步
- 主要兑现类型（信息/能力/关系/资源/认可/情绪）：信息兑现 + 情绪释放

## 2) 催化事件（Catalyst）
- 事件：封不觉值第一个夜班，"不说话的患者"推门进来，留下一件遗物后离开——触摸遗物时，死者的记忆灌入他脑内
- 不可逆变化：灵视被完全激活，从此无法关闭——诊室在午夜0-3点成为两界交界点，封不觉被绑定为"执念完成者"
- 主角当下目标：搞清楚诊室发生了什么，确认自己不是疯了

## 3) 升级危机链（Fichtean 危机递增，至少 3 次）
| 节点 | 危机/冲突 | 代价/风险升级 | 结果/变化（可量化优先） |
|------|-----------|---------------|--------------------------|
| 1 | 第一案：自杀女孩的遗书——封不觉第一次接触灵体，死者的绝望情绪淹没他 | 精神被污染1-3天，出现失眠、噩梦、死者的情绪在体内残留 | 完成第一个执念（帮女孩把没说出的话传递给母亲），获得第一块非童年记忆碎片，验证"完成执念=释放灵体"可行 |
| 2 | 搭档登场——前刑警带着血浸笔记本出现，自称"已死之人"，笔记本里有封不觉七岁的照片和"第六面，已确认"字样 | 搭档要求封不觉帮他藏身，诊室成为两人的共同秘密。封不觉意识到自己的过去不是"忘记"而是"被操作" | 获得搭档作为盟友+信息源，但也继承了搭档的敌人。诊室从"奇怪的工作地点"变成"藏身所" |
| 3 | 第三/四案：怨念灵体开始主动"找上门"——越来越多的灵体感知到诊室的存在，封不觉无法拒绝任何一个 | 污染累积加速（失眠恶化、开始做死者的梦），搭档警告"你在消耗自己"但封不觉无法对灵体说不 | 建立诊室运作模式（白天普通实习+午夜灵媒），但污染已从"偶尔不适"变为"持续症状" |
| 4 | 组织第一次接触（第30章）——调查员以"心理学会督导"身份来访，问了不该问的问题 | 搭档认出对方是组织外围人员，诊室已被标记。二人必须决定：继续还是逃？ | 选择继续——因为封不觉不能丢下正在处理的执念。搭档开始准备逃生方案，诊室从"安全区"变成"倒计时的安全区" |
| 5 | 第五/六案中途，封不觉的污染症状被外人注意到——诊所主任怀疑他"精神状况不适合工作" | 双重暴露风险：诊所内部（可能被开除→失去诊室）+ 组织外部（能力波动被探测器捕捉→组织距离缩短）| 搭档首次替封不觉做决定——未经同意伪造了"心理咨询记录"应付主任。二人第一次激烈争吵 |

## 4) 中段反转（必填）
- 假胜利/假失败/反转事件：封不觉成功完成第五个执念，获得一块异常清晰的记忆碎片——但这块碎片不属于死者，属于他自己七岁那年。记忆里有人在说"第六面，确认完成"——封不觉意识到自己不是"天生灵媒"，是被"制造"出来的
- 反转带来的新认知/新代价：诊室不是他偶然入职的地方——他被安排在这里。搭档的笔记本上"第六面，已确认"不是记录他，而是记录一个实验。二人从"猎人与猎物"变成"实验品与实验者"
- 若无：-

## 5) 卷末最低谷（All Is Lost）
- 最低谷事件：组织确认诊室位置，派出灵媒特工直接接触。搭档为了给封不觉争取完成最后一个执念的时间，主动暴露自己行踪引开追兵——搭档被捕/失踪。封不觉独自在诊室完成执念，但完成后发现搭档没有回来
- 代价：搭档失去联络（生死不明），诊室的安全窗口期结束（组织已知位置），封不觉的污染从中度逼近重度
- 主角选择：不逃——留在诊室等搭档回来。同时，在完成第六个执念的瞬间，记忆回流——他看见七岁的自己站在一扇门前，门上写着"第六面"

## 6) 卷末大兑现 + 新钩子（Payoff + Next Promise）
- 本卷兑现（对应 1) 开卷承诺）：
  1. 六个执念案件的完整释放——读者已理解"完成执念"的情感重量
  2. 搭档身份的完整揭示——笔记本内容部分解密，组织追杀原因初步明朗
  3. 组织从"暗处威胁"到"正面逼近"——压迫感兑现
- 新钩子（引入下一卷承诺）：搭档被捕后，组织传来消息——"想要他活着，封不觉必须来'第七面'。"什么是第七面？诊室之外还有另一个交界点？
- 章末未闭合问题（落到最后一章）：搭档是生是死？封不觉七岁时在"第六面"门前看到了什么？组织说的"第七面"在哪里？
