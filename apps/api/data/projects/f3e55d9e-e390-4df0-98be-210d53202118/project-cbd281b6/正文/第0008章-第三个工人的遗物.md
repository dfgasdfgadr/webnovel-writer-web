# 第0008章 第三个工人的遗物

封不觉回到诊室的时候，午夜刚过。
走廊灯暗了一半，空调不知什么时候停了。
空气变干变薄，像被抽走了什么。
指腹蓝痕还在发热。
从小卖部出来到现在，温度就没降过。
残留比他预想的浓得多。
他推开值班室的门。
工人灵体站在诊室中央。
比昨晚更清楚了。
工作服上的水泥斑点、裂成两半的安全帽从头顶延伸下来的裂缝、喉咙位置灰白色的堵塞物——全都看得分明。
灵体没有动。
它只是站在那儿，面对着门。
像是在等他。

封不觉关上门，在桌前坐下。
安全帽还在原处，帽檐朝着他。
他把左手放在安全帽旁边，指腹朝下，没有立刻触碰。
"你昨晚给我看了你是怎么死的。"
灵体没有反应。
"但不是全部。"
指腹落在帽檐上。
蓝痕与安全帽表面接触的瞬间——眼前是荧光灯。

嗡嗡响的荧光灯管钉在低矮的天花板上，照得整间办公室像泡在消毒水里。
一张铁皮办公桌。
桌上摊着两张打印纸——欠条。
金额栏的数字写得很大，五位数。
一个不该出现在这种纸上的数目。
桌对面坐着人，但记忆里看不清脸。
一只胖手压在欠条边缘，指甲缝是黑的。
工人的手在发抖。
粗糙的指节握着笔，笔尖在签名栏上方悬了很久。
那只胖手敲了敲桌面。
"签。你不签，你家里人——"
后面的话被办公室墙外的打桩声吞掉了。
笔落下去。
字写得歪歪扭扭。
纸被抽走，一边被撕——签名那半张，被人收进了抽屉。
剩下的半张被工人攥在手里。
攥到纸缘嵌进了掌心。

封不觉睁开眼。
额头全是汗。
指腹蓝痕烫得像要烧穿皮肤。
他低头看自己的左手——手指还是他自己的，但刚才握笔的触感还钉在骨节里。
他把安全帽翻过来，拇指沿着衬布边缘一寸一寸地按压。
右下角。
那里比别处厚了一层。
缝线已经脆了，稍一用力就断了两针。
衬布翻开。
帽壳和衬布之间夹着一张折叠成方块的纸，被汗浸过又被时间烘干，纸缘已经脆化。
封不觉展开它。
半张欠条。
金额栏的数字清晰——一个工人不吃不喝攒五年也还不上的数目。
签名那栏被整齐撕掉。
撕口是直的，像是用尺子比着撕的。

灵体向前移了一步。
诊室温度骤降。
灯管闪了一下，荧光粉的余晖在黑暗里拖了半秒。
安全帽的金属扣上凝出一层薄霜。
灵体张开嘴。
喉咙里的水泥裂缝中挤出一个字。一个人能听懂的字——清晰的，不带杂音。
"钱。"

封不觉把残片平摊在桌上，用手掌压平卷起的边角。
抬头看灵体。
"你被迫签的。"
灵体的轮廓剧烈地晃了一下。
是形态本身的震颤，像信号不好的电视画面。
诊室温度又降了一截，封不觉呼出的气变成了白色。
灵体抬起手臂，手指指向残片，停了两秒。
然后手臂弯曲，做了一个向下压的动作。
缓慢的，强制的。
像有人按住它的肩膀把它压下去。
喉咙里的水泥裂缝张开。
"逼。"
声音比"钱"更费力，水泥裂缝在发音时往外挤出了细碎的灰粉。
手臂维持着被按住的姿势。
然后猛地向前一送——手掌推出去。
推一个人。
推一扇门。
推一个站在高处的人。
"推。"

诊室灯管彻底灭了，只剩下走廊透进来的半截暗光。
灵体收回手臂，站在原地。
轮廓边缘在发抖。
封不觉拉开抽屉，取出值班记录本，翻到最后一页——没有印刷格线的空白页。
笔尖在纸上停了两秒。
写下：
"钱——被迫签欠条（记：五位数，工人五年工资总和）"
"逼——办公室里，对方不露脸，指甲缝是黑的"
"推——签完被灭口。三个人都是被推/被砸/被触电=不同死因，同一个模式"
"三人同工地，互不认识。同一笔债务。债权方可能是凶手。"
他放下笔。
"另外两个人——也签了同样的欠条。"
陈述句，没带问号。
灵体没有动，但轮廓边缘的震颤停了一拍。
那是一个点头。

封不觉把欠条残片夹进记录本，合上。
指腹蓝痕的温度开始回落。从沸点退到了热，还没退到常温。
他抬头。
灵体站在诊室中央。
它转过身，面朝窗户。
手臂抬起来，食指贴在玻璃上。
路灯把玻璃映成一块模糊的灰色。
灵体的手指在玻璃上没有留下指纹、没有水汽、什么都没有。
喉咙里的水泥裂缝张开的幅度比先前任何一次都大。
灰白色的堵塞物被从内部撑开，干燥的水泥粉末簌簌落下。
一个字。
没有杂音，没有工地噪音垫底。
干净的，清楚的。
"血。"

窗外什么都没有。
路灯下的街道是空的，树是黑的，对面的楼没有亮灯。
封不觉听到了一个声音。
第三个人临死前的声音——他从未亲耳听过。
一口气。
从喉咙挤出来的，最后的。
他低头看自己的手指。
蓝痕还在发热。
白线还是那条白线。
但耳朵里那个声音没有消失。
像锤击声，隔着混凝土墙，一下一下地敲。
封不觉把记录本放回抽屉。
窗外还是什么都没有。
灵体的食指还贴在玻璃上。
