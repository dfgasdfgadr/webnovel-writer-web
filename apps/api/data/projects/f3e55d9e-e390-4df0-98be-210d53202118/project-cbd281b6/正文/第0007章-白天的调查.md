# 第0007章 白天的调查

下午两点。
封不觉站在工地对面的一排商铺前。
围挡后面，吊臂在空中缓慢旋转。
没有人在施工。
指腹的蓝痕微微发热。
残留比灵体更轻、更散，覆盖着整个工地入口。
那顶安全帽的主人曾在这里站过走过活过。
他推开小卖部的玻璃门。
门上的风铃响了一声。
柜台后面一个五十来岁的男人抬头看了他一眼，手指还夹着烟。
封不觉扫了一眼柜台。
一份旧报纸摊在玻璃柜台上，头版标题对着他——「XX工地再发事故，一人死亡」。
日期是两周前。
他拿起报纸，翻过来看背面。
另一条更小的新闻被压在角落，同一个工地，三个月前的。
他把报纸放回柜台。
"老板，这工地离这儿不远吧。"
老板弹掉烟灰，看了一眼他身上的白大褂。
"不远。走过去五分钟。"
烟在指间转了一圈。
"你是医生吧——怎么跑工地来了。"
封不觉指了指报纸。
"有个病人住这附近，路过来看看。"
指腹在柜台边缘轻轻压了一下，蓝痕的温度又升了半度。
"最近这工地出过事？"
老板把烟叼回嘴角，从柜台下抽出一张更旧的报纸，已经泛黄了。
"你是第三个来问的人。"
他把两张报纸并排放在柜台上——一张两周前，一张三个月前。
"前两个——一个说是记者，一个说是律师。"
烟灰掉在柜台上。
"记者后来没再见着。律师——"
他停住。
"律师也没再见着。"
封不觉没有说话，指腹的蓝痕温度升高了一度。
视线从两张报纸上移开，落在柜台下面。
老板把第三张报纸压在柜台下。
没拿出来，但封不觉看到了一角，日期更早。
"不止这两个。"
他抬眼看着老板。
"还有几个。"
老板把烟头按进烟灰缸，看了封不觉一眼。
目光越过白大褂，停在他的眼睛上。
"那个工地啊——这半年死了三个人了。"
他用三根手指按住柜台上的报纸。
"都是意外。"
第三根手指——食指，关节粗，指甲里嵌着灰白色的水泥。
"报纸就登了一个。另外两个连豆腐块都没有。"
他收回手，从烟盒里又磕出一根。
"工人家属签了保密协议，拿了钱就不说话了。"
顿了顿。
"不过你要是问工地的人，他们会告诉你——不是意外。"
封不觉把手从柜台上拿开。
"三个人，分别是什么时候。"
老板划燃打火机。
"第一个，五个月前。掉下来的。"
"第二个，三个月前。触电。"
火光照了他脸一下，灭了。
"第三个，两周前。被砸的。"
封不觉记下了。
指腹的蓝痕持续发热，温度稳定在一个低频的灼烧感上。
残留比他预想的要浓得多。
"他们都住附近？"
老板摇了摇头。
"不住。都是外地来的。"
他把烟点上，朝工地围挡的方向抬了抬下巴。
"人没了——东西还在工棚里。公司说等月底统一清理。"
封不觉推门离开小卖部。
风铃又响了一声。
阳光照在工地围挡上，吊臂还在转——缓慢，持续，像某种不停止的东西。
指腹的蓝痕温度没有降下来。
他站在围挡外面，看着工棚的方向。
老板说东西还在里面。
三个人，三个月内的东西。
那个安全帽只是其中之一。
他掏出手机。
地图上，诊室到这里五公里。
组织距离缩短的里程——如果他再完成一个执念，会再近一截。
他把手机放回口袋。
先找到另外两个人的遗物。
然后，再决定要不要在诊室里碰它们。
