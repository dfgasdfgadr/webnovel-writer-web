# 第0002章 死者的遗言

封不觉跪在地上，膝盖磕得发麻。
手机屏幕上的时间是零点四十一分。
他低头看了一眼手指——指腹上的蓝色墨痕还在。
他用拇指擦了两下。没掉。
又用纸巾蘸了水擦，纸巾上是皮肤摩擦的红痕。
墨痕纹丝不动。
他把手指举到灯下。
蓝色不在皮肤表面。在下面。

起来的时候，膝盖真的青了。
他弯腰揉了揉，把地上的日记本捡起来搁到桌上，后退两步坐到转椅上。
封面墨绿色，边角磨得发白。
封不觉盯着它看了大概有二十秒，然后伸手把它合上了。
盖上之后，他等了等。
什么都没发生。
又翻开。
第一页。蓝色圆珠笔的字迹，笔压很轻。
封不觉没有碰页面边缘——只压着书脊，让纸页自然摊平。

第一行：今天开学。
字迹圆圆的，横轻竖重不均匀，写的人用力很轻。
第二行：分到六班，窗边。可以看到操场。
第三行：同桌叫何梦瑶，人很好，她说这里的食堂比初中好吃。
封不觉往下读。
讲数学老师说话有口音。
讲体育课跑八百米跑得肺疼。
讲放学路上有一家奶茶店第二杯半价，她一个人喝了两杯。
一篇很普通的开学日记。日期是九月二号。
他翻到下一篇。

日期跳过了一周。九月九号。
她说被选进学生会宣传部了。
前排有个男生话很多，每次都转过来借橡皮。
笔调还是轻的，但字迹比第一篇潦草了一点。
有几个字被划掉了——笔尖反复画的横杠，不是修正带。
封不觉能辨认出被划掉的是什么。
不是用眼睛看的——指腹的蓝痕在碰到那行划痕的时候疼了一下，像针扎进指甲缝。
一个名字。姓王的男生。
九月十六号。名字没有再出现。
九月二十三号。日记只有三句话：
今天没去食堂。放学后留在教室改板报。回家天黑了。
九月三十号。
课间去厕所，听见里面有人笑。
笑她校服的衣角是破的——她之前从来没发现那里有线头。
她把线剪了，把剪刀放在笔袋里。
封不觉翻页的手指停了一下。
日记没有写年份，只标了日期。
他数了数页数——本子用了一半左右，九月到十月中旬的日记还在，十月中旬以后有几页被撕掉了。
不是自然脱落，撕得很深，连装订线里都残留着纸根。

十一月的第一篇日记，笔迹变了。
用力很重，笔尖把纸压出凹痕。
第一行写：她们把我的书从三楼扔下去了。
第二行写：不是不小心掉的。是等在那里的。
封不觉把日记本合上了。
他的手在抖。不厉害，但从手指尖一路抖到手腕——像那条蓝色墨痕的位置。
他把右手按住左手，深呼吸。
吸到一半停下了。
想起来这是他今晚第几次深呼吸。
然后刻意把这口气吐掉。
"低血糖？"
他对自己说的。
值班室抽屉里有一包饼干，他拆开来吃了两片。
甜的，胃里有东西落下去，但手还在抖。和血糖没关系。
"喝了什么？"
饮水机的水，傍晚倒的。杯底没有异味。
他拿起来又喝了一口——正常。没被下药。
"没睡好？"
到诊所之前在出租屋睡了六小时。深度睡眠，没有做梦——至少昨天还没有。
他把饼干袋放下，把值班记录本翻开，在上面写了一行：零点四十一分，日记一本，来源不明，无法确认为幻觉。
写完以后他自己看了看这行字。
一个心理系的实习生，在值班记录上写"无法确认为幻觉"——这件事本身就不像是精神正常的表现。
他在下面又补了三个字：但手不抖了。

然后他把日记本重新翻开。
一张折叠的小纸条从十一月和十二月之间的夹缝里滑出来，落在桌上。
从练习本上撕下来的横格纸。封不觉把它展平。
上面只有四个字：不是你的错。
字迹被水渍或泪痕洇过，圆珠笔的墨迹晕开了一点点。
这张纸条没有写给谁，也没有署名。
夹在一本没人该看到的日记里。
封不觉没往下分析。
他把纸条原样折回去，翻到日记最后一页。
末页没有日记正文。
只有一行电话号码。
还有一串字，被指甲刮掉了。
他凑近看，想辨认笔画——纸面已经被刮毛了，蓝墨陷进纤维断口里，什么也拼不完整。

他抬头想揉眼睛。手举到一半，停了。
诊室角落站着一个人。
封不觉没有站起来，没有喊，没有碰任何东西。
他的眼睛停在她身上，手指尖还压着日记的末页。
页码的边角被指腹的汗濡湿了一小块——这是接下来大概十秒内唯一在动的东西。
角落里的女孩穿着校服。蓝白配色，长袖，袖口有点脏。
头发扎起来了，几根碎发散在耳朵前面。
她的嘴唇在动——翕开一小道缝，又抿上，又翕开，像在念一个很长的词。
没有声音传过来。
诊室的空气变了。
刚才还有空调的嗡声，现在什么也没有。
连封不觉自己的呼吸声都听不到了。
像一个透明的罩子扣下来，把所有振动都隔绝在外面。
"你能听见——"
封不觉自己的声音被从空气中掐掉了。
说到第三个字的时候，他的喉咙还在震动，但耳朵里一个字都没有。
某种力量把声音从空气里抽走了。
他说了"你能听见我说话吗"，只听到了前两个字的轮廓——然后空气抹掉了后面所有。
女孩看着他，摇了摇头。
那个摇头的意思很清楚：别费劲了，说不出来的。
然后她抬起右手。
用食指在空气中从左向右画了一道线——在自己左手手腕的位置，横切过去。
动作不重。
像在指出一个事实。

封不觉低头看日记末页的电话号码。
末页的数字写得很整齐——一笔一划，和前面越来越潦草的日记完全不同。
有人在最后一次翻开这个本子的时候，用最认真的字迹把一个电话留了下来。
他看了那个号码大概五秒，没有拿手机。
然后他抬头。
女孩的轮廓开始变淡。
不是消失——是密度被空气稀释了。
像一层水汽被什么东西从边缘往里吃掉，一寸一寸变薄。
她的嘴唇已经不翕动了，最后一个动作是指了指封不觉的左腕——不是她自己的，是他的。
她散掉了。
封不觉呼出一口气。
气变成白雾——一团实打实的白汽飘到他面前，然后消失。
诊室温度不高。
他身后的饮水机温度计显示了二十四度。正常室温。
他又呼一口：又是白雾。但只有呼向诊室角落方向的时候才有。

然后空调嗡的一声重新响了。
走廊的回声回到正常长度。
他听到饮水机咕咚冒泡、窗外后巷有野猫踩了铁皮棚。
这些声音之前被什么东西筛掉了，现在一口全吐了出来。
他站起来走到门口，握了一下门把手——凉了。和傍晚一样凉。
凌晨三点。

封不觉转回来面对桌子。
日记本还是翻在末页，电话号在灯下没有任何异常了——墨迹就是墨迹。
他拿起手机。解锁了屏幕。翻到拨号界面，手指停在半空中。
那是明天该想的事。
锁屏。放下手机。
松手的瞬间袖口落回去，左手腕内侧一凉。
一道浅青色的痕。
不是撞的，不是磕的，是皮下血管的颜色但分布不均匀——集中在一条线上，从左到右，正好和女孩在空气中画的那道位置相同。
他用手擦一下，又擦一下。
和指腹上的蓝色墨迹一样。不在皮肤表面。擦不掉。
他拿起值班记录本，在刚才那行"无法确认为幻觉"下面，另起一段。
笔尖停顿了很长时间，纸上洇了一个墨点。
然后写了三个字：青痕。手腕。左手。
