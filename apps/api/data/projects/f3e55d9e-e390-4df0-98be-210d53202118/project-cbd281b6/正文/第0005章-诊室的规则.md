# 第0005章 诊室的规则

"它到底是什么？"
封不觉把结晶举到晨光下。六面体，半透明，内部嵌着极细的丝状纹路。
结晶比刚出现时小了三分之一。
他放下来，翻开值班记录本。最后一页写着五条规则——昨晚睡前写的，字迹有些歪，但都编了号。他重读了一遍。拿起笔。在第五条下面补了一行。
6. 结晶持续缩小。原因不明。
合上记录本。食指侧面，蓝痕还在。旁边是两道白痕——极细，平行排列，像被什么极薄的刀刃划过。
昨天只有一道。第二道是今早出现的。他在指腹上按了一下，不疼。放下手。穿白大褂。去大厅签到。
白天他进出值班室好几次。每次挂钟的指针都在不同位置——九格、十一格、八格。没有规律。秒针一直在走。他记下浮动范围。等午夜。
中午路过五号诊室。门开着——周医生在接诊，对面坐着一个中年男人。桌上放着一杯水。封不觉多看了那杯水一眼。水面是平的。
晚上十一点半。他坐在值班室里。结晶放在桌面上，正对挂钟。手机屏幕朝上。
空调低频嗡鸣。他把空调关了。嗡鸣消失，安静变大。远处走廊偶尔传来一声闷响——管道热胀冷缩，或别的什么。他把那声音也等过去。
封不觉站到挂钟前。
指针在第十一格。离十二还差半个指甲的距离。
他盯着那根黑色的分针。看了很久。久到眼睛发干。
他眨了一下眼。
指针的位置不一样了。
指针直接变了——从第十一格变成了第十二格。中间没有移动。
他低头看手机。零点整。
秒针停了。时针和分针叠在一起，钉死在十二点位置。他盯了整整五分钟。纹丝不动。
他下楼。大厅挂钟在走——零点零五分，秒数正常跳动。手机同步。
他回到值班室。指针仍锁在第十二格。
他在记录本上写道：午夜零点起，诊室挂钟指针锁定于第十二格。大厅挂钟与手机时间不受影响。秒针停走。
他放下笔。又拿起来。又放下。
把结晶握进手心。
画面叠在现实上面——两张底片同时曝光。
食堂。中午。靠窗的位置。女孩坐在左边，何梦瑶在对面。女孩把饭盒里的红烧肉夹到何梦瑶碗里。何梦瑶抬头看了她一眼。女孩笑了一下，嘴唇在动。在说什么。没有声音。持续了大约四秒。消失。
封不觉低头看手里的结晶。比刚才小了一圈——用尺子量，直径少了接近一毫米。
太阳穴开始发紧。钝疼。像有什么东西在颅骨内侧逐渐收拢。他把结晶放进抽屉，用拇指按压太阳穴。没用——紧的位置在更深的地方。
他低头看手。食指蓝痕没有变化。白痕仍是两道——第二道比早上更长了，从第一指节往上延伸了小半厘米。在皮肤下面。像某种沉积。
他站起来，走出值班室。
走廊空着。日光灯的色温偏冷，地板上的浅灰塑胶面反出一层薄光。五号诊室的门关着。他把手背贴上去——木门，常温。
他走回值班室。在门口停了一步。握住门把手。
凉的。金属在冬天不供暖时的那种温度。他把手拿开，等了五秒，又握上去。还是凉的。
推开值班室的门——室内的空气比走廊凉。温差也许只有一度。皮肤能分辨。
他在记录本上画了一道竖线分隔。写上"连通态"三个字。旁边注明：午夜零点起，门把手温度低于走廊，室温略降。挂钟锁止。
凌晨一点半。他拧开水龙头。自来水冲过手指上的蓝痕——无反应。
擦干手。从抽屉里拿出结晶。指尖碰上去。刺痛从指腹传到手腕。
他把结晶放回去。又碰了一次。一样。
凌晨两点。指针仍钉在第十二格。
封不觉坐在椅子上。等。
空调没开。走廊的闷响也停了。整栋楼被塞进一个大号隔音罩。他自己的呼吸声变得极清晰——吸气时鼻腔里轻微的哨音，呼气时嘴唇分开的声音比平时响两倍。
两点十五分。安静太久了。过度安静。像什么东西把所有声音都吸走了。胃部在收紧。紧绷——一根极细的线从胃底穿过去，往两边拉。
他把记录本翻到新的一页。笔握在手里。没写字。
凌晨两点四十分。
敲门声。
三下。
指关节叩击木门。不重，不轻。间隔均匀——笃。笃。笃。
封不觉没有站起来。结晶在抽屉里。记录本摊开。钟的指针锁在第十二格。手机屏幕朝下扣在桌上。他没有问"谁"。门没有再响。
