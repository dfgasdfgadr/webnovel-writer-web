# 世界观设定

## 世界一句话
表面是现代都市，但里世界的灵体通过执念物品渗入生者世界——第六面诊室是两界唯一的稳定交界点。

## 题材融合点（复合题材可填）
- 主题材：悬疑灵异（恐怖氛围、灵异现象、执念灵体）
- 副题材：侦探推理（案件调查、逻辑推演、刑警式办案）
- 融合机制（1-2句）：每个灵异案件是一起"只有灵媒能调查的悬案"，用刑警式调查方法+灵媒能力双线推进。灵体提供只有死者知道的线索，但接受线索的代价是情感污染

## 世界结构
- 大陆/位面数量：双层——现实都市 + 里世界
- 核心区域：
  - 现实世界：某现代都市（待命名），普通人对灵异现象无感知
  - 里世界：灵体存在的"另一层"，由人类执念和负面情绪凝结而成，不可直接进入
  - 交界点：第六面诊室（午夜0点-凌晨3点两界重叠），其他交界点不稳定且危险
- 边缘区域：其他临时交界点（凶宅、事故现场、执念极强的灵体周围），不稳定，随时可能关闭

## 势力格局
- 核心势力：隐藏组织（待命名）——知晓里世界的秘密组织，拥有灵媒成员，利用灵体执念结晶作为能源/信息源。对封不觉的态度：初期观察，中期利用，后期清除
- 次级势力：普通人类社会——对里世界一无所知。但组织渗透在社会的各个层面（医疗、公安、政府）
- 敌对/中立关系：
  - 组织 vs 封不觉：敌对（长期），组织视诊室为战略资源，视封不觉为"野生灵媒"需要收编或清除
  - 组织 vs 搭档：死敌（组织杀了搭档的搭档，搭档掌握了组织核心秘密）
  - 灵体 vs 组织：灵体是组织的"资源"，组织是灵体的"猎人"
- 宗门/组织层级：组织的结构——上层（决策层，身份不明）→ 执行层（灵媒特工，负责接触/捕获灵体）→ 外围（普通人，提供后勤/掩护/社会层面的操作）

## 社会结构
- 社会阶层：现代都市职业阶层。普通人对里世界无感知，但组织的渗透意味着"表面正常的都市，底层有看不见的战争"
- 资源分配规则：执念结晶——灵体被"完成"后遗留的晶体，含有灵体的记忆碎片和情绪能量。组织收集结晶作为力量来源。封不觉的诊室是执念结晶的天然收集点
- 信仰/意识形态：组织内部可能有一套"为了人类"的意识形态——利用灵体是为了研究里世界、保护人类社会不被灵体侵蚀

## 历史年表（关键节点）
- 约25年前：封不觉出生，"第六面"交界点首次被记录
- 约18年前（封不觉7岁）：封不觉目睹童年惨案，记忆被"取走"。同一事件也是组织的起源/一次重大事故
- 约10年前：前刑警搭档开始调查灵异案件，逐渐发现组织存在
- 约3年前：搭档被组织发现并追杀，社会记录被改为"死亡"
- 现在：封不觉入职心理诊所，诊室作为交界点开始活跃

## 核心规则
- 资源稀缺性：稳定的交界点极其稀缺（已知只有第六面诊室），组织一直想控制一个稳定交界点。执念结晶是稀有资源——每个灵体只产生一颗
- 政治/宗门规则：组织在暗处行动，利用社会体系掩盖灵异事件（将灵异死亡伪装成意外/自杀/失踪）
- 社会常识/禁忌：普通人无法感知灵体，但灵体可以影响普通人的情绪和梦境。长期暴露在灵体附近的人会出现"污染症状"（抑郁、幻觉、记忆混乱）——这被社会当作精神疾病
- 硬约束（不可违背）：
  1. 灵体不能被消灭，只能被"完成"（执念达成后自然消散）或"囚禁"（组织的方法）
  2. 灵媒体质不可逆转——一旦觉醒就无法回到"看不见"的状态
  3. 交界点有严格的时间窗口——第六面诊室只在午夜0-3点连通
  4. 执念结晶的读取是不可逆的——读取后结晶消失，记忆进入读取者

## 复合题材融合逻辑（可选）
- 规则兼容点：悬疑灵异的世界观天然兼容侦探推理——死者是最佳证人，但也是最大陷阱（死者的记忆可能不完整/被扭曲/被污染）
- 冲突点与解决：推理需要逻辑，灵异不一定讲逻辑。解决方式：灵体本身不讲逻辑（情感驱动），但执念的形成有逻辑（每个人死于非命的背后有可推理的真相）
- 副题材介入条件：每个案件的前1/3是灵异体验（恐怖氛围+遗物读取），中1/3是调查推理（搭档主导+线索串联），后1/3是执念完成（封不觉主导+情感释放）
- 详见：设定集/复合题材-融合逻辑.md

## 世界运转机制
- 能量/资源循环：人类负面情绪和未完成的执念 → 在里世界凝结成灵体 → 灵体通过执念物品与生者世界接触 → 执念被完成 → 灵体消散留下结晶 → 结晶中的能量可被灵媒吸收/利用
- 技术/法术基础：灵媒体质是天生的（或通过特定方式"觉醒"），非灵媒无法通过技术手段感知里世界。组织在研究"人造灵媒"——这可能是封不觉七岁时遭遇的事件的背景
- 公平性与代价规则：每接触一次里世界，污染就加深一步。没有"安全地使用灵媒体质"的方法——保护自己的唯一办法是不用。但不用就无法帮助灵体。这就是世界的残忍之处

## 货币与经济
- 货币体系：现实世界为现代法定货币（人民币），组织内部可能有"结晶交换"体系
- 兑换规则：执念结晶是硬通货——不同强度的执念产生不同强度的结晶
- 主要流通形态：在普通人世界不可见。组织内部通过结晶交易获取力量/信息/延长灵媒寿命
- 交易场景：组织内部的黑市/交易所，普通人不参与

### 常见货币模板（可选）
- 现实世界为现代货币体系，与普通人无异
- 里世界层面不存在"交易"，只存在执念的"交换"——封不觉帮灵体完成执念，灵体留下结晶（可能包含封不觉需要的记忆碎片）

## 地理版图
- 重要地点清单：第六面诊室（核心舞台）、搭档的安全屋（城市边缘废弃公寓顶层，后因组织追查废弃）、组织总部（未知位置）、城中各处案件现场（每案不同地点）
- 关键资源点：诊室是唯一稳定交界点=最关键的战略资源。其他临时交界点（凶宅、废弃医院等）是次要资源点
- 组织设施：废弃安全屋（城郊废弃仓库地下——存有物证箱和组织照片墙）、组织废弃数据节点（可被入侵获取实验档案）

## 时间规律（第一卷揭示）
- **午夜0-3点**：诊室作为交界点的标准连通时间
- **满月夜**：交界点达到峰值——诊室墙壁半透明可见里世界轮廓，灵媒能力被动放大（污染效率×3），组织灵媒在满月夜行动力最强
- **诊室钟规则**：钟停在午夜12格——是嵌入里世界的女孩"按了暂停"。每完成一个灵体执念，分针走一格。走完一圈（12格）=门会开=封不觉可以与女孩重新建立联结

## 执念结晶分类（第一卷揭示）
- 白色（淡白→白）：悔恨类执念——释放感
- 灰色：怨念类执念——真相被揭露后平息
- 蓝色（淡蓝）：等待类执念——被动等待多年的释放
- 暗红色：怨恨/正义类执念——不是"释然"而是"够了的平静"
- 黄色：存在类执念——"被记住"的温暖
- 深灰色：愧疚类执念——保护者"我已经尽力了"的被原谅
- 深紫色：真相类执念——"知道了答案"的释然
- 淡绿色：传承类执念——不是痛苦，是"把东西传下去"
- 执念结晶用途：①读取灵体记忆；②共鸣补全被删除的记忆；③布置"执念场"（混合灵体记忆的区域结界）；④制造信号干扰器（临时遮蔽诊室灵异信号，只能维持数小时）

## 组织技术与世界观交互
- **执念场**：将多颗执念结晶嵌入特定方位，利用共振形成混合灵体记忆的区域。进入者被多种执念情绪同时冲击。在诊室内因满月夜共振效果最强。被封不觉用于防御（第62章）。原理：完成执念产生的"释放"能量比组织强制抽取的"囚禁"能量更完整——二者相遇时释放能量占优
- **反灵结界**：组织技术装置，四个装置同时激活→形成高频灵体噪音屏障→阻断灵体进入与灵媒能力使用。需四装置同时破坏才能关闭。可用于"封锁"诊室类交界点
- **工具人**：组织消耗品——用执念结晶控制普通人，体内承载数十个被囚禁灵体的痛苦。释放后工具人昏迷但可存活（灵体离开身体）。体现了组织"不在乎人命"的核心逻辑
- **韩铎的中立**：组织内部并非铁板一块——部分灵媒成员意识到自己被当消耗品，寻找脱离/自救的方法。韩铎是其中最理性的一个：不背叛组织（因为需要结晶资源续命），不效忠组织（因为知道自己会被用完即弃），保有自己的目标（找那个女孩）

## 日常生活
- 货币/交易方式：正常现代社会
- 交通/通信：现代通信（但搭档用一次性手机/暗号联系以避免追踪），组织可能监控通信
- 教育/职业体系：现代都市职业体系。封不觉白天是普通实习生，午夜后是灵媒——双重生活是持续的压力来源
