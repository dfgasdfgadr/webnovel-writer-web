## 主角卡

### 姓名

林逸

### 性格

林逸是一名25岁的程序员，性格内向但观察力敏锐，具有强烈的正义感和好奇心。他平时沉默寡言，但在关键时刻能表现出惊人的果断和创造力。他讨厌不公，但更倾向于用逻辑而非暴力解决问题。由于童年经历，他对“记忆”和“真相”有执着的追求。

### 能力

- **初始能力**：Lv.2 搜索者，灵能插件为“猎犬”，擅长深度搜索和记忆读取。
- **特殊天赋**：浏览器UI界面异常稳定，能同时打开3个标签页（同级只能开1个），且扩展程序兼容性强。
- **主要技能**：
  - **标签页切换**：能在“分析模式”“隐身模式”“攻击模式”间切换。
  - **书签锁定**：保存关键信息或技能，即使失去能力也能快速恢复。
  - **历史回溯**：看到目标地点或人物过去的数字活动痕迹（每天限用2次）。
- **成长路线**：
  1. 初期：利用搜索能力揭露小型犯罪，积累经验和扩展程序。
  2. 中期：加入“秩序浏览器”或“自由浏览器”阵营，学习更高级的编辑和操控技能。
  3. 后期：发现数字帷幕的真相，挑战“根权限者”或“AI意识体”，并决定是否改写现实规则。

### 背景故事

林逸在新京市“锈带”区长大，父母在他10岁时死于一场离奇的车祸，但警方报告显示是“系统错误”导致的自动驾驶故障。他自学编程，成为了一名成功的自由开发者。22岁时，他在调试一个涉及量子加密的项目时，意外觉醒为浏览器。他逐渐发现，父母的死可能与数字帷幕中的某个隐藏势力有关。为了寻找真相，他开始深入探索浏览器世界。

### 核心欲望与缺陷

- **核心欲望**：查明父母死亡的真相，并揭露数字帷幕背后的阴谋。
- **缺陷**：过度依赖逻辑，有时忽视情感因素；容易因为好奇心而陷入危险；对“真相”的执着可能让他忽视身边人的安全。
- **成长主题**：从孤独的搜索者成长为负责任的守护者，学会在真相与安全之间找到平衡。
