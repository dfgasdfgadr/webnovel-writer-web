## 细纲（第 8 章）

### 第 0008 章：第三个工人的遗物

- **核心事件**：封不觉从白天调查回到诊室，再次接触安全帽读取深层记忆——工人死前一周被迫签下巨额欠条；安全帽内衬夹层里藏有欠条残片，工人灵体开始能说简单词语。案件从"可能谋杀"升级为"经济纠纷谋杀"。
- **字数目标**：2800 字

#### 情节点序列

1. 封不觉从小卖部回来，回到诊室——蓝痕温度未降，残留比预期浓。
2. 工人灵体已在诊室等待——比昨晚更清晰，能发出断断续续的单字。
3. 封不觉再次触碰安全帽，进入更深层记忆：死前一周，工人在办公室被迫签下巨额欠条。
4. 记忆中的细节——办公室逼仄、欠条上的金额、签字时手在发抖。
5. 封不觉从记忆中脱出，检查安全帽内衬——发现夹层，里面有欠条残片（金额清晰，债权人签名被撕掉）。
6. 工人灵体用简单词语回应——"钱""逼""推"——确认经济纠纷是动机。
7. 封不觉推断三个工人都被同一笔债务牵连，凶手可能在债权方。
8. 章末：封不觉准备离开诊室，工人灵体突然清晰说出"血"，指向诊室窗外。

#### 目标情绪
白天调查后的疲惫 → 深入记忆的压迫 → 欠条发现的突破 → 工人开口的冲击 → 章末"血"的不安。

#### 章首钩子
场景切入 —— 回到诊室，蓝痕还在发热。工人灵体站在诊室中央，比昨晚更清晰。

#### 爽点
信息兑现——欠条残片=经济动机确认；工人灵体首次说出可辨识的词语（"钱""逼""推"）。

#### 章尾钩子
隐藏含义钩——工人灵体突然清晰地说出一个字："血。"然后指向诊室窗外。窗外什么都没有。但封不觉听到了——不是窗外的声音，是记忆里的。第三个死者临死前的声音。

#### 涉及角色
- 封不觉（POV）
- 工人灵体（第二案灵体——进步：能说单字）

#### 本章变化
- 案件从"可能谋杀"→"经济纠纷谋杀"（欠条残片）
- 灵体从"说不清话"→"能说简单词语"（执念推进=灵体清晰度提升）
- 污染加深：从幻视扩展到幻听（工地锤击声）
- 欠条金额巨大，债权人签名被撕——凶手可能来自债权方
