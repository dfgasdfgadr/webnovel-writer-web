# 总纲

## 故事一句话
心理诊室是生与死的交界——实习医生封不觉能看见死者的遗物，每件遗物是一个未完成的执念。唯一知道这个秘密的搭档，是一个在社会记录中"已死亡"的前刑警。每完成一个执念，封不觉离童年惨案的真相近一步，也离追杀搭档的隐藏组织近一步。

## 创意约束
- 反套路规则：AT-004（反派共享同一能力——隐藏组织也有灵媒，但选择利用而非帮助）+ AT-009（每次"正确选择"带来负面后果——完成执念暴露信号，组织逼近一步）
- 硬约束（世界/能力/行为）：
  1. 封不觉不能主动关闭灵视——看到就是看到，逃不掉
  2. 灵体不能被消灭，只能被"完成"——每个案件的解法是完成死者的执念
  3. 每次使用灵媒体质，组织的探测器离诊室近一公里——能力使用=暴露风险
- 主角缺陷：过度共情——感知死者痛苦如己出，界限模糊，容易被"污染"（情感污染→人格渗漏）
- 反派镜像：隐藏组织与主角共享灵媒能力。主角道路=帮助死者完成执念、释放灵魂；反派道路=利用灵体获取力量/信息，将死者当作资源

## 核心主线
- 主线目标：找回封不觉童年目睹惨案后丢失的记忆，揭开"不说话的患者"的真实身份
- 主要阻力：每一次调查都在暴露自己（组织在逼近）；每一次接触灵体都在污染自己（人格渗漏在加深）；童年记忆不是"丢失"而是"被取走"——有人在阻止他想起来

## 核心暗线
- 暗线主题：记忆可以被取走，但执念不会消失——封不觉帮助死者完成执念的过程，也是他自己被取走的记忆通过"执念"的方式一点点回流的过程
- 回收节点：每卷末回收一块童年记忆拼图，最后一卷揭示完整真相——不说话的患者与封不觉童年惨案的关系

## 反派分层（概要）
- 小反派（案件级）：执念灵体——未必是恶，是未完成的执念扭曲成的存在。解法是"完成"而非消灭
- 中反派（组织级）：隐藏组织"（待命名）"——拥有灵媒，知晓里世界，利用灵体榨取力量。追杀前刑警搭档，盯上封不觉的诊室作为交界点
- 大反派（存在级）：不说话的患者——出现在诊室的神秘存在，与封不觉童年惨案相连。每完成一个案件，"患者"就更接近"说出真相"

## 世界观/力量体系简介
- 世界观要点：单城（现代都市）+ 里世界（灵体所在的"另一层"），第六面诊室是生与死的交界点（午夜0点-凌晨3点连通）。灵体通过执念物品与主角建立联系。
- 力量体系要点：污染等级体系——轻度（能看见）→ 中度（能触摸）→ 重度（能对话）→ 不可逆（人格渗漏，混淆自己与死者的记忆）。不是"升级"而是"污染加深"，每加深一步都是不可逆的选择。

## 复合题材融合（可选）
- 主题材 / 副题材：悬疑灵异（主题材）/ 侦探推理（副题材）
- 融合机制（一句话）：每个灵异案件是一起"只有灵媒能调查的悬案"，用刑警式调查方法+灵媒能力双线推进
- 节奏占比（建议 7:3）：恐怖氛围/灵异体验 7，推理调查/逻辑推演 3
- 关键冲突与爽点：案件真相揭露（爽点=执念完成的释放感）、组织对抗（爽点=反猎杀的紧张智斗）、记忆回流（爽点=拼图合拢的恍然大悟）

## 主角成长线
- 起点状态：心理诊所实习医生，刚获得职位，以为是普通工作。灵媒体质尚未完全显现（不知道自己能看见的是什么）。
- 关键跃迁节点：
  - 第一次"看见"（诊室第一夜，意识到自己不是幻觉）
  - 第一次"触摸"（污染中度，能主动接触灵体了——但代价立刻显现）
  - 第一次"对话"（污染重度，能听到死者生前最后的话——但开始混淆记忆）
  - 最终选择（人格渗漏临界点——继续使用能力会失去"自己"，但不用就无法完成最后一个执念）
- 终局定位：取决于最终选择——可能成为诊室永久的管理者（守在交界点），可能完成所有执念后灵媒体质消失（回归普通人），可能完全渗漏成为里世界的一部分
- （多主角）分工与冲突：
  - 封不觉：灵媒能力+心理专业，负责接触灵体、读取执念、理解死者
  - 前刑警搭档：调查能力+组织内幕，负责线索追踪、逻辑推演、掩护封不觉
  - 冲突点：搭档倾向于"藏起来不被组织发现"（保守），封不觉倾向于"完成执念哪怕暴露"（冒险）。两人对"什么值得冒险"的判断不断碰撞

## 关键爽点里程碑
- 第1章：诊室第一夜——不说话的患者出现，第一件遗物出现，第一段死者记忆被读取
