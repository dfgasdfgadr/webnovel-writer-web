# 主角组设定（双主角）

## 主角列表
| 名称 | 定位 | 主线贡献 | 关键缺陷 | 关键能力 |
|------|------|----------|----------|----------|
| 封不觉 | 灵异案件的灵魂解读者 | 接触灵体、读取执念、完成释放。主线暗线（童年记忆回流）的唯一载体 | 过度共情——界限模糊，容易被污染，无法对灵体说"不" | 天生灵媒体质（灵视、触物读念、灵体对话） |
| 前刑警搭档（待命名） | 案件的逻辑推演者 + 组织线枢纽 | 案件调查、线索串联、掩护封不觉、组织内幕信息。主线明线（组织对抗）的核心推动者 | 过度保护——因上一个搭档被组织杀害，对封不觉的保护近乎偏执，会替封不觉做决定 | 刑警训练的观察力/推演能力、地下人脉网络、组织内幕知识 |

## 共同目标
- 团队共同目标：完成灵体的执念（短期）、扳倒组织（中期）、揭开"不说话的患者"和封不觉童年惨案的真相（长期）
- 阶段性目标：
  - 第一卷：在诊室建立安全运作模式，完成第一批执念，搭档身份初步揭示
  - 第二卷：组织正式介入后，在躲避追捕的同时继续完成执念
  - 第三卷：揭露组织起源与"不说话的患者"的关联
  - 第四卷：终极选择——封不觉的人格完整 vs 最后一个执念

## 角色分工
- 决策者：封不觉——关于灵体的事（"我们要完成这个执念"），搭档——关于安全的事（"太危险了，换条路"）。冲突时最终决定权取决于"谁更有道理"而非固定权威
- 执行者：两人各执行自己擅长的——封不觉负责灵体接触，搭档负责人间调查
- 信息中枢：搭档掌握组织信息和调查人脉；封不觉掌握灵体层面的信息（记忆碎片、情绪线索）
- 情感支点：搭档是封不觉在人间唯一的锚——提醒他"你是谁"。封不覺是搭档唯一的"值得保护的人"——这是搭档活下去的意义

## 视角与节奏
- POV 分配比例：封不觉 POV 70%（沉浸式灵异体验、执念完成、记忆回流），搭档 POV 25%（调查线、组织线、保护者的焦虑），其他 POV 5%（偶尔的灵体POV/反派POV，用于揭示隐藏信息）
- 轮换规则：同一场景只选一个 POV——两人同时在场时，选"信息更少"的那个 POV 以保持悬念。搭档 POV 章通常设在封不觉接触灵体后——读者从搭档视角看到封不觉的变化（"他今天说话的语气不像他自己"），恐怖感来自旁观者的视角
- 防止抢戏的约束：搭档 POV 章不出现灵异体验（他看不见）。封不觉 POV 章不出现组织全局信息（他不知道）。两人各有"只有他能看到的世界"

## 内部冲突
- 价值观冲突："值得为陌生人（灵体）冒多大风险？"——封不觉的过度共情让他无法拒绝，搭档的保护欲让他想拒绝所有高风险案件
- 资源冲突：时间——封不觉想完成越来越多执念，搭档需要更多时间调查组织和躲避追踪。二人对"先做哪件事"永远在拉扯
- 信任裂缝：搭档隐瞒了上一个搭档是怎么死的（因为涉及封不觉七岁时的事件），封不覺隐瞒了自己污染加深的程度（因为不想搭档担心而阻止他）

## 团队成长弧线
- 组建期（第一卷）：被迫合作——封不觉需要搭档的调查能力和组织信息，搭档需要封不觉的诊室作为藏身点。初期互不信任但不得不信任
- 磨合期（第二卷）：冲突爆发——组织逼近后，保护 vs 冒险的价值观冲突激化。第二卷会有一次"分开"（搭档被迫逃亡，封不觉独自面对案件），各自意识到缺了对方不行
- 裂变期/重组期（第三-四卷）：信任建立——搭档开始尊重封不觉的选择（"我不是在保护一个需要保护的人，是在支持一个知道自己在做什么的人"），封不觉开始对搭档完全坦诚（包括污染的真实程度）

## 与创意约束对齐
- 反套路规则如何影响团队：AT-009（正确选择带来负面后果）——每次封不觉完成执念，搭档的藏身处就更危险一步。二人在"做正确的事"和"保护彼此"之间永远在拉扯
- 硬约束如何迫使协作：灵体只能完成不能消灭——需要搭档的人间调查来找到执念的解法。灵视不可关闭——需要搭档来分辨"封不绝看到的是真的还是污染产生的幻觉"。组织逼近——需要封不觉的诊室作为唯一安全区

## 第一卷关系推进实录
- **组建期（第15-28章）**：搭档登场（第15章）→ 第一次分工商量（第16-17章）→ 第三案合作完成（第18-19章）→ 笔记本揭示（第20章）→ 正式结盟但有保留
- **磨合期开始（第29-40章）**：组织第一次接触后二人对"是否撤离"的分歧（第31章）→ 搭档伪造心理咨询记录（第39章）→ 第一次激烈争吵："你不能替我做决定"（第39章核心冲突）→ 冲突后冷却（第40章：封不觉选择睡在安全屋=信任底层未变）
- **磨合期深化（第41-56章）**：第六案双灵体——封不觉用"共同完成"解法（第45-46章）→ 中段反转后封不觉质问搭档隐瞒实验信息（第48章）→ 信任裂缝公开但继续合作 → 第七案搭档补全"挑选标准"答案（第56章）
- **裂变前奏（第67-72章）**：满月夜清除组来袭 → 搭档主动引开追兵（第67-68章："你留在这里——让我做完"）→ 搭档被捕成为人质（第70-72章）→ 封不觉接受组织条件保证搭档活着（第72章）——第一个"替对方做决定"的对称反转：第39章是搭档替封不觉，第72章是封不觉替搭档

## 第二卷关系起点
- 搭档被扣押在组织内（"活着但人质"状态），每三个月交换一次存活证据
- 封不觉在组织的"协议"下继续诊室工作——表面接受条件，暗地每完成一个执念让钟走一格
- 二人的信任裂缝依然存在：封不觉瞒着搭档"我接受了条件但暗地在找那个女孩"，搭档瞒着封不觉"上一个搭档可能是组织的卧底"（第64章种子）
- 第二卷的核心关系张力：搭档从"保护者"变成"被扣押者"——当二人重新取得联系时，权力关系已经变了
