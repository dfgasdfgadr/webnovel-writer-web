# 主角卡

## 基本信息
- 姓名：封不觉
- 年龄：25岁
- 身份：心理诊所实习医生（刚入职）、天生灵媒（隐藏）
- 起点状态：第一天值夜班，尚未完全意识到自己的灵媒体质——看见的东西以为是幻觉

## 核心标签
- 3个关键词：灵媒、过度共情、失忆者
- 读者第一印象：一个自己满身裂痕却还在帮别人修补碎片的人

## 性格与底色
- 核心性格：安静、观察力强、内向但非社恐——能在诊室里从容面对患者，但离开诊室就是另一个人
- 行为底线：不能接受"利用他人的痛苦"——这是他与隐藏组织的根本分歧
- 情绪触发点：看到无助的人/灵体会本能地想去帮助（过度共情的正面表达）
- 易激怒点：被人当工具利用（因为他自己一直在被"某种力量"利用而不自知）
- 容易心软点：对孩子的执念——因为自己的七岁记忆是空的

## 动机与目标
- 短期目标：搞清楚诊室发生了什么，确认自己不是疯了
- 中期目标：完成第一个灵体的执念，验证"释放"是否真的有效
- 长期目标：找回童年（七岁那年）丢失的所有记忆
- 真正渴望（可能不自知）：不是"记起过去"，而是确认"自己是谁"——一个人没有童年记忆就没有来处

## 缺陷与代价
- 性格缺陷：过度共情——感知死者痛苦如己出，界限模糊，容易被"污染"
- 能力限制：灵视不可关闭（看到就是看到），灵体只能"完成"不能消灭
- 心理阴影：童年惨案后的记忆空白——七岁那年的一切是黑的，但身体记得恐惧
- 代价承受底线：可以接受自己受伤，不能接受搭档/他人因为自己被组织捕获

## 关键关系
- 重要盟友：前刑警搭档（唯一知道秘密的活人，调查能力+组织内幕信息源）
- 主要对手：隐藏组织（也有灵媒，但利用灵体）、不说话的患者（既是线索来源也是最终谜底）
- 情感关系：无感情线
- 关键债务/牵绊：搭档救过封不觉的命（第一次接触灵体失控时），封不觉欠他一条命——这让每次"用能力→组织逼近"都意味着背叛搭档的信任

## 镜像对抗
- 共享欲望/缺陷：隐藏组织的灵媒也能看到灵体——他们和封不觉一样"被选中"，但选择了利用而非帮助
- 反派道路：组织将灵体当作资源（执念结晶是能源/信息源），封不觉将灵体当作"未完成的病人"

## 当前能力
- 境界/等级：污染轻度——能看见灵体、能通过遗物接收死者的情绪残片
- 代表技能：触物读念（触摸遗物可感知死者最后的情绪/执念，代价是情绪在身上停留1-3天）
- 资源/装备：第六面诊室（交界点，午夜0-3点连通里世界）、搭档的血浸笔记本（组织内幕碎片）

## 金手指
- 类型：天生阴阳眼/灵媒体质
- 代价/限制：情感污染→人格渗漏。每次接触死者执念，死者情绪在体内停留数天。污染加深：轻度（看见）→中度（触摸）→重度（对话）→不可逆（混淆自己与死者的记忆，开始说死者说过的话）
- 核心卖点：能力是负担而非礼物——每一次使用都让读者为他感到疼

## 行为模式
- 常用解决方式：理解（尝试理解灵体的执念是什么）→ 共情（感受死者的痛苦）→ 完成（找到执念的解法，释放灵体）
- 失败时的本能反应：不是愤怒或放弃，而是"再来"——过度共情让他不能接受"我帮不了你"
- 破局特长：用心理医生的思维对待灵体——不是驱鬼，而是"治疗"。问他"你为什么在这里"而非"你怎么还在这里"

## 成长弧线（阶段）
- 阶段1（起点）：逃避——不想承认自己能看到灵体，想假装是普通实习生。搭档的登场强迫他面对
- 阶段2（变化）：接受——从被动卷入到主动调查，开始把灵体当作"患者"而非怪物。但污染在加深
- 阶段3（蜕变）：抉择——当人格渗漏逼近临界点，他必须在"失去自己"和"放弃最后一个人"之间选择。如果选择"完成最后一个执念"——代价可能是永远回不来，但这就是他成为"封不觉"的全部意义

## OOC 警戒
- 绝不该做的事：漠视灵体的痛苦（"这不关我事"）、利用灵体获取利益（那他就和组织没区别了）、在搭档需要他时退缩
- 需要提前铺垫的事：七岁那年目睹了什么（碎片式闪回）、灵媒体质是天生的还是"被植入"的、诊室为什么选他来当实习医生（不是巧合）
