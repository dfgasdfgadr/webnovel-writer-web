# 第0001章 入职第一夜

封不觉到诊所的时候，天还没黑透。
接待大厅比招聘简章上的照片小一圈。
米色墙面，深蓝色候诊沙发。
茶几上搁着一盆绿萝，叶子擦过，土是新换的。
一个三十多岁的女人坐在前台后面翻挂号本，抬头看了他一眼。
"封不觉？"
"是。"
她从抽屉里抽出一张访客卡，推过台面。
"主任在二楼，最里面那间。卡刷电梯。"
封不觉接过卡，往电梯间走。
身后那女人补了一句："向右拐。左边走廊不通——六号诊室在那一头，封了。"
封不觉回头。
她已经低头继续翻本子了。
大厅两侧各一条走廊，灯都亮着。
左边那条的尽头有个拐角，距离太远，看不清拐过去是什么。
他没多问，向右转了。

主任姓陈，五十出头，头发梳得整齐，说话慢。
他把实习合同、值班表、诊室分布图一字排开在桌上。
"你主要在五号诊室跟周医生。周医生周四有门诊。"
翻过一页纸。
"其余时间在档案室和值班室——值夜班。每周两次。没病人的时候，整理病历。"
封不觉点头。
陈主任的目光在他脸上停了一拍，比正常的注视久了那么一点。
"有什么想问的？"
"夜班到几点？"
陈主任把手里的笔帽拧上。
"早上八点。不过——过了午夜，就不要在各楼层走动了。记录可以第二天补。"
他把笔搁下。
封不觉没追问。
实习生第一天上班，问太多不是好事。

下楼的时候，前台那女人正在打电话，声音压得低。
封不觉只听到半句——"……今年的实习生。"
她看见封不觉从电梯里出来，把话筒捂住，给他指了值班室的方向。
值班室在二楼，隔壁是档案室，对面是五号诊室。
没看见六号诊室的门牌。
封不觉在二楼转了一圈。
从电梯口走到走廊左端，再走回右端。
五间诊室。
没有第六间。
他重新看了一遍分布图。
图上标了六间。
傍晚六点到八点之间，诊所还在运转。
有病人进出，有医生在走廊里接电话。
候诊沙发上的盆栽摆得规规矩矩。
封不觉在档案室里归置病历，一个文件夹一个文件夹地翻。
格式统一，墨迹工整。
但他注意到一件事。
所有病历的建档日期都在过去三年之内。
没有更早的。
他没问。

八点以后，病人走完了。
九点半，最后一位医生刷卡出门。
十点整，前台的灯灭了。
封不觉一个人待在值班室。
值班室不大，一张桌子、一把转椅、一台旧电脑、一台饮水机。
窗外是后巷，路灯把电线杆的影子投在对面的砖墙上。
他从抽屉里翻出值班记录本。
上一页有人写了"一切正常"四个字，日期是一周前。
再上一页也是"一切正常"。

十一点半的时候，他去了一趟洗手间。
走廊的灯还亮着。
走过去的脚步声和傍晚不一样了。
每一步的回音都在拐角后面找不到落点，走廊比他想象的长。
路过楼梯口时，他往下看了一眼。
大厅的黑沉在底层，和二楼走廊的亮隔着一道看不见的面。
经过五号诊室门口，门把手上没有反光。
他站住了。
门把手是金属的，走廊灯就在头顶，应该反光。
他把手伸过去——手指停在把手前方一厘米的位置。
没有凉意。
空调房里，金属把手不该是常温的。
他收回手，回了值班室。
把门关上了。
门没有锁。
他想了想，没有锁。

凌晨差五分十二点。
走廊尽头传来一声响，像旧木板被什么东西踩了一脚。
然后是第二声。
封不觉放下手里的病历，站起来。
那声音停了。
停了大约三秒。
值班室的门他关上了，他确定。
但刚才那一瞬间，门框里的黑暗和门板之间的那条缝——宽度变了。
封不觉把手放在门把手上，没有转动。
门把手是凉的。
和五号诊室那把不一样。

午夜零点。
走廊另一头有东西被推开了。
声音从他傍晚走过的右边尽头传来，隔了不止一层——不在这一层，在左边走廊封住的那一端。
声音被拐角吃掉了一部分，但封不觉听到了。
一扇门。
开了。
脚步声——软底鞋，步伐不快，没有犹豫。
封不觉盯着门，右手还在把手上，指尖的汗和金属表面粘在一起。
声音停了。
在他这扇门外面。
封不觉把手从把手上拿开，退了一步。
门下沿的光带，被两道影子遮住了。
是一个人的两只脚。
"谁？"
没人回答。
手机还亮着，屏幕上是十分钟前刷到一半的文章。
他按灭屏幕。
空调停了。
墙壁的隔音好得不正常——走廊的声音一滴都渗不进来。
但他能感觉到那个人还站在门外，隔着一层木板和一个没有锁的把手。
大约过了五秒，门缝下的影子退了。
脚步声往回走，还是那个速度，不快不慢。
踩在地砖上，被走廊一截一截吞掉。
封不觉没有听到任何一扇门重新关上的声音。
他等了十秒，把门拉开。
走廊空无一人。
灯亮着，间距均匀，没有一个灭的。
地面干净，没湿痕没脚印。
他走到走廊尽头往左看。
左翼拐了过去，看不清拐角后面有什么。
电梯正对着他，显示的楼层是当前楼层。

封不觉走回值班室。
经过五号诊室时，他试了试门把手——凉的，和傍晚一样凉。

他回到桌前，坐下，拿起刚才没喝完的水。
纸杯的杯底落在桌上时，桌上多了一样东西。
一本日记本。
封面是墨绿色的，边角磨得发白。
封不觉盯着它看了三秒。
他站起来去开门的时候，桌面上只有病历和值班记录本。
这本子刚才不在这里。
他回头看了门口。
走廊还是亮的，宽度和傍晚一样。

他翻开日记本。
第一页。
字迹均匀，蓝色圆珠笔，笔压很轻。
封不觉没有读完第一行。
翻开封面的瞬间，纸页的纤维间涌出一股东西。
从指尖灌进去——不在眼睛，不在耳朵，直接进了意识里。

天台。
傍晚。
火烧云烧满了西边整面天空。
脚趾在鞋里蜷起来。
站在天台边缘的时候，风是往下的。
两只脚尖伸出天台边，悬空了。
楼下是操场跑道，有人在跑步。
有人在喊一个名字，隔着远。
然后脚底空了。
风灌进耳朵。
时间是慢的，慢到能看清每个楼层窗口里亮着的灯——从楼顶往地面数，以坠落的速度。
什么也抓不住。
喉咙里没有尖叫。
不敢叫。
叫了这件事就变成真的了。
快要碰到地面的时候，心里还可以喊一声什么。

封不觉睁开了眼睛。
他跪在值班室的地上。
窗外的路灯还亮着，照着对面同一面砖墙。
手机屏幕上的时间是零点四十一分。
日记本摊开在他手边，第一页被他的手指按住的地方，墨迹渗进了指腹。
他低头看着自己的手指。
指腹上有一小道蓝色的痕。
怎么擦都擦不掉。
