# 第0004章 残留

封不觉醒了。
不是被闹钟吵的。是自己醒的。
窗帘缝里没有光——天还没亮。他摸到手机，四点半。
睡了多久算不清。但这是他这几天第一次自己醒来，不是被噩梦惊醒，也不是被走廊的声响吵醒。
他躺着没动。
左手腕摸了一下。青痕还在，但更浅了。像铅笔写淡了的线。
右手指腹的蓝痕碰上去没有反应。
他把手举到眼前，在暗光里看。
蓝痕还在。但颜色有没有变——判断不了。
他坐起来，开了床头灯。
日记本在床头柜上，结晶在旁边。
小小一粒，灯光底下是半透明的乳白色。
他把结晶拿起来，对着灯泡看。
里面有一点光在流转。不是真的在流——是光线透过不同密度时弯了一下。
他把它放回去。
去洗手间，拧开水龙头。
冷水冲到脸上的时候，有一瞬间不对劲。
他抬头看镜子。
脸上是水珠，眼眶下还是有黑影，眼球有一点点血丝——不是熬夜的那种红，是更细的，像针尖划的。
他靠近镜子，看着自己的瞳孔。
瞳孔是正常的。黑色的。
但他盯着看了有好几秒。
他总觉得瞳孔里面有什么东西。
不是影像。是位置不对——瞳孔里的反光点，比应该有的偏了一点。
他关了灯。再看。
瞳孔在暗处放大了。
里面有三个人影。不是他背后的——是里面的。
他后退一步，打开了灯。
镜子里只有自己。眼眶微青，头发乱着。
他站在镜子前又看了几秒，然后用毛巾擦干脸。
走到厨房，倒了一杯水。
水是凉的。胃里有东西，不饿。
但喉咙发紧。不是感冒——没鼻塞没咳嗽。是胸口的某根筋，一直绷着。
上午九点他到诊所。
前台女人在翻挂号本。抬头看他，没说话。但目光多停了一下。
封不觉走进值班室。
桌子上还是昨天的病历，记录本打开着。
他在桌前坐下，翻开记录本。
昨晚写的最后一行是"第一个"。
他拿起笔，在下面加了一行：第四天。睡眠正常。瞳孔有异常反光。不影响视力。
上午的病人不多。
封不觉在档案室整理病历。一个文件夹一个文件夹地翻。
有一个老年患者，姓刘，七十多岁。
从档案室出来的时候，封不觉在走廊里看见了他。
刘老坐在候诊沙发上，手里拿着挂号单，腿并着，腰背挺得很直。
他身后有一团影。
不是影子——影子在地上的，那团东西在他肩膀后面，飘着。
模糊的，形状像一个人缩成很小一团，又像一捧没化开的烟。
封不觉站住了。
刘老回头看了他一眼。"医生，到我了吗？"
"还没有。"封不觉说。他的视线没法从那个影子上移开。
"您一个人来的？"
"一个人。"
封不觉点了点头，转身走回档案室。
他把门关上，手按在墙壁上。
指腹的蓝痕碰到了墙——没反应。
刚才那团灰色影子不是灵体。或者不是完整的灵体——没有面目，没有动作，只是在刘老肩后飘着。
而且刘老不知道自己身上有东西。
他深呼吸一下。然后回到值班室。
刘老被叫号的时候，封不觉在值班室门口看着。
老人站起来，影子也跟着站起来。
速度一样。方向一样。
像一个正常的影子——但不在正确的位置上。
中午，封不觉在值班室里吃饼干。
甜的在嘴里化开。他拿出结晶，放在桌上。
小小一粒。
他用拇指把它按在桌面——不硬，像一粒盐。
然后握进手心。
指腹蓝痕碰到结晶——一针的疼。
然后画面涌进来。
不是女孩站在天台边缘。
是她坐在教室里。
下午。阳光从窗户斜进来，把桌子腿的影子拉得很长。
她在写日记。
笔尖压在横格纸上，一笔一划，蓝墨在纸上洇了一点。
她写到"同桌叫何梦瑶"，停了一下。用笔杆敲了一下下巴。然后继续写。
她笑了。
不是在对着谁笑。是想到什么了。
封不觉把结晶放回桌面。手心是空的。
画面的残留还在眼眶后面——阳光的颜色，笔压在纸上的声音。
他没有哭。不是悲伤。也不是不悲伤——是别的。是"被允许"的感觉。她笑了。
他把结晶重新握紧，继续看。
画面跳了一下。
女孩在天台上。傍晚。火烧云占了半边天。
她没有站在天台边缘——坐在天台中间的水泥地上，抱着膝盖。
在等人。不是等死。
是等人来跟她说"不是你的错"。
没有人来。
她站起来。走到天台边缘。
没看她下面是哪里。只是看着天。
封不觉松手。结晶从手心滚到桌上。
他拿起水杯喝了一口。水是冷的。
笔不知道什么时候握进了手里。指节发酸。
他低头看手心里的结晶——比刚才小了一点。
不是错觉。是真的小了一圈。
他拿尺子量了一下——约一毫米缩了。
下午刘老回诊室拿药。
封不觉在走廊里看到他的时候，他叫住老人。
"刘老，能跟您聊两句吗？"
老人停了一下，然后点头。
"您最近——有没有睡不好？"
"有一点。"
"做梦多吗？"
"做了，记不住。但醒来——"老人停了停，"醒来特别空。"
"空了？"
"就是觉得少了什么。明明梦里有东西，但一睁眼就没了。"
封不觉看着他肩膀后面的那团灰影。
"您老伴还好吗？"
老人沉默了好一阵。
"她走了三年了。"
封不觉点头。"对不起。"
"没事。"老人转头看向窗外。"你说奇怪不——三年了，有时候还觉得她在这间房间里。"
灰影动了一下。不是飘。是颤。
封不觉把手按在桌面上。
指腹的蓝痕没有反应。
但这个老人没有遗物在诊所——他是活着的患者，不是灵体。为什么他的记忆会"附着"过来？
他低头看自己的手腕。青痕的位置。有一点点痒。
下班前，他去了一趟主任办公室。
主任在打电话。看见他进来，捂住话筒。
"有事？"
"想问一下，诊室有没有过——以前的值班记录？"
"值班记录都是每个人自己记的。我这里没有存。"
"那之前值班的——"
"前一个实习生干了不到一个月就走了。"主任说。"去年的事。"
"走了之后呢？"
主任放下话筒。"你的意思是想看他的记录？"
封不觉说："随便问问。"
主任看了看他，没有追问。只是说："如果有旧记录本，应该在地下室仓库。钥匙在保洁阿姨那儿。"
封不觉点了点头。
他没有去地下室。
他回到值班室，在椅子上坐下来，把结晶放在抽屉里。
然后翻开自己的值班记录本。
翻到最新一页。
在上面写了今天的内容：刘老，灰影。不是灵体——是生者残留。诊室白天也在工作。
他合上本子，看着窗外。
后巷的路灯亮了。野猫从铁皮棚上跳下来，踩了一声白铁皮的闷响。
白天的诊室和晚上的诊室是同一个空间。
但它的"功能"在白天没有关。
只是强度不一样。
午夜是开水龙头。白天是水龙头没拧紧——一滴一滴渗。
他拿起记录本，把之前写的规则重新写过。
第一条：午夜零点到凌晨三点，诊室是生者与死者的交界。
第二条：灵视在第一次接触遗物后开启，无法关闭。
第三条：遗物承载死者记忆，触碰可读取。
第四条：完成执念=灵体释放。释放后遗留执念结晶。
第五条：诊室白天也是薄交界点。能看到生者的残留——未完成怀念、太想留住的人。
他把"太想留住的人"划掉，改成"未完成的关系"。
凌晨十二点，他躺在值班室的折叠床上。
身上盖着外套。眼睛闭着。
脑子里有一个声音在转。
不是女孩。是他自己的。
在重复同一句话："第一件，完成。"
然后是下一句，不知道是谁的声音，但他能听出来不是自己的——"那第二件呢？"
他睁开眼。天花板的日光灯熄灭着，只有窗外路灯漏进来的光。
他坐起来。
拿起结晶放在手心。还剩原来的大概三分之二。
如果握久了会变小——那变小了是什么意思？能量释放了？消耗了？
他把结晶放回抽屉。合上抽屉的瞬间，指腹蓝痕疼了一下。
不是刺痛。是那种被什么东西按了一下。
他低头看手指。蓝痕在暗光里是深蓝色的。
和他第一天看到的颜色一样。
但旁边——又多了一道。
不是青痕。不是蓝痕。
是一道很细很细的白痕。在食指内侧。像是刚刚被划的。
没有出血。不疼。但能看到。
他把手指放到灯下。
三条痕迹：蓝、青、白。
三条线。三个颜色。三种标记。
他盯着看了很长时间。
然后把手指握进掌心，攥了一会儿。
温度正常。
他躺回去。
闭上眼睛。十五岁女孩的笑脸在黑暗中闪了一下。
然后定格了。
不是她站在天台边缘的画面——是她坐在教室里，笔杆敲着下巴，笑着写日记。
这个画面不疼。
是可以留在脑子里的那种。
六点。天亮了。
封不觉从值班室出来，走到走廊尽头。
五间诊室。没有第六间。
他站了一会儿，然后走回值班室。
把记录本收进抽屉。
结晶在抽屉最里面，隔着一层木头，温度透不上来。
但他知道它在。
他把抽屉关上。
新的夜班——明天晚上。
白天，他还得做一个普通人。
但他透过走廊的窗子看出去——候诊沙发上已经有人坐着了。不是病人——是来的最早的病人。她肩膀上坐着一团透明的影。
封不觉看了一秒。两秒。
然后转身回了值班室。
今天不处理。
今晚不值班。
今晚——他需要想清楚一件事。
他把结晶放进裤袋。
走出诊所大门。
三月的早晨有一点薄雾。路边梧桐树的枝丫在雾里看不清方向。
他走过街角。
站台上有三个人在等车。
一个老太太。一个穿校服的中学生。还有一个中年男人。
三个人身后都有影。
不是灰色的不是黑色的——是透明的。像热气从地面升起来。
封不觉看着他们。
公交来了。他上车。刷卡。走到最后排坐下来。
然后是下一站。上来的人。身后也有东西。有的浓，有的淡。
他把手插在口袋里。拇指和食指捏着结晶。
小小一粒。
全城的人，很多人身后都带着东西。
他之前看不见。
现在看见了。
