# 第0003章 第一个执念

封不觉是被闹钟吵醒的。
不是手机的闹钟，是隔壁租客的老旧挂钟。
咚咚咚敲了七下。
他睁开眼，天花板还是那片泛黄的水渍。
右手腕内侧一凉。
那道青痕还在。从左到右，一条细线，颜色比凌晨浅了一点。
但还在。
他用拇指擦过去。没有凹凸感，像画在皮肤下面的。
指腹上的蓝色墨痕也还在。
和昨天一样，擦不掉。
他坐起来，把枕头靠在床头。
日记本搁在床头柜上。墨绿色封面，边角磨白。
他盯着它看了大概十秒，然后伸手拿过来。
直接翻到末页。
电话号码还在。一行数字，蓝色圆珠笔，一笔一划写得整齐。
旁边是被指甲刮掉的痕迹，纸面毛糙。
他把手指按在那串数字上。
指腹的蓝痕没有反应。不疼。
封不觉把手机从枕头底下抽出来。
屏幕亮了一下，电量百分之四十三。
他打开拨号界面，一个数字一个数字地输入。
输到最后一位，手指停了。
他不知道自己是谁。
一个陌生男人，给一个失去女儿的母亲打电话。
说"有人让我告诉你"——这话说出来，像诈骗。
他按下删除键，一个数字一个数字删回去。
然后重新输入。这次输完，没停，直接按了拨号。
电话响了四声。
第五声之前，接起来了。
"喂？"
是一个女人的声音。四十多岁，沙哑，带着刚睡醒的疲惫。
封不觉的喉咙紧了。
"您好。"
对面没说话。可能是在等他开口，也可能在辨认这个号码。
"有人想让我转告您一句话。"
"谁？"女人的声音变了一点。警惕的。
封不觉看着手腕上的青痕。
"我不知道该怎么说。"
这是实话。
"您认识一个叫——"他顿住了。
他不知道女孩的名字。
日记本上没有写名字。第一页只有"今天开学"四个字。
他翻过前面的页码。没有署名。
"我不知道她叫什么名字。"封不觉说。
"但我有一本日记。墨绿色封面。蓝色圆珠笔写的字。"
电话那头沉默了。
沉默持续了大概三秒。或者五秒。封不觉数不过来。
然后他听到一声吸气。很短，像被什么东西扎了一下。
"你是谁？"女人的声音在抖。
"我——"封不觉低头看自己的手腕，"我在诊所值班。这本日记出现在我桌上。我读了。"
"你读了什么？"
"我读到一个女孩。九月开学，分到六班。同桌叫何梦瑶。"
念一句，停一下。电话那头没有声音。
"她被欺负了。十一月，书被人从三楼扔下去。"
"够了。"女人的声音突然打断他。
不是生气。是崩断了。
封不觉没说话。
他听到电话那头有动静。像是手捂住了话筒，或者手机被按在了什么布料上。
然后有声音传过来。压抑的，闷在手掌心里的，哭声。
他握着手机，指腹的蓝痕开始疼。
像针，一下一下扎进指甲缝。
他低头看。那道蓝色的痕迹比刚才深了一点。
不是光线的问题。
哭声持续了一阵。然后话筒被重新拿起来。
"她最后一晚，"女人的声音碎成几片，"我们在吵架。"
封不觉听着。
"她说'都是我的错'。我说'不是'。她就跑了。"
女人吸了一口气。那口气吸到一半变成了哽咽。
"我追出去。电梯关了。我跑楼梯。十三层。"
她停了一下。"我到天台上的时候，已经——"
封不觉说："她不是想说'都是我的错'。"
电话那头安静了。
"她让我告诉您——不是您的错。那天晚上，她是想说这个。"
他不知道自己为什么要用"她让我"这种措辞。
但说出来的时候，指腹的蓝痕不疼了。
像那根针被拔掉了。
女人在哭。不是嚎啕，是那种一口气吊不上来的哭法。
封不觉不知道该怎么办。
他从来没在电话里听一个陌生人这样哭过。
"她还说了什么？"女人问。
声音很远，像是把手机放在桌上，人蹲在一边说话。
封不觉翻开日记，找到那张纸条。
"有一张纸条。夹在日记里。四个字。"
他念出来："不是你的错。"
电话那头没有回应。只有哭声。
封不觉说："对不起。"
然后他挂了电话。
不是不想听。是不知道该听什么。
他盯着手机屏幕看了大概二十秒。
通话时长：三分十七秒。
他把手机扔到床上。
右手按在左手腕上。青痕的位置。
皮肤下面有什么东西在跳。不是脉搏。是别的。
他去洗手间洗脸。
冷水冲在手上的时候，他抬头看镜子。
眼睛下面有黑影。但脸色正常。不苍白。
他关了水龙头。水滴在池子里，一滴，一滴。
像有人在数。
上午九点半，封不觉到了诊所。
前台的女人在翻挂号本。抬头看了他一眼。
"脸色不太好。"
"没睡好。"
他穿过走廊，走到值班室。
门开着。桌子上有病历，有值班记录本。
他坐下，翻开记录本。
最后一行是他凌晨写的：青痕。手腕。左手。
他盯着这五个字看了几秒。
然后在下面写：上午九点四十七分。电话已打。
白天的诊所是正常的。
病人进出，医生在走廊里打电话，候诊沙发上的绿萝叶子擦过扶手。
封不觉在档案室整理病历。
一个文件夹一个文件夹地翻。
但他的注意力不在上面。
他的右手一直在摸左手腕。那道青痕。
每次摸上去，皮肤下面都有一点跳动感。
像有什么东西在回应他。
中午他在便利店买了饭团。
坐在 clinic 门口的台阶上吃。
太阳很好。三月的阳光不烫，但亮。
他咬了一口。米饭是冷的。
然后眼泪就下来了。
不是因为他想哭。是他的眼睛在流泪。
没有悲伤的感觉。没有情绪。
就是眼泪在往外流。
他用手背擦了一下。又擦了一下。
对面路过的人看了他一眼，走开了。
他低头看饭团。咬过的缺口上有牙印。
他把饭团吃完。眼泪也一直在流。
流到吃完最后一口，停了。
他站起来，把包装纸扔进垃圾桶。
左手腕上的青痕比上午深了一点。
他用手遮了一下，进了诊所。
下午没什么事。
他在值班室坐了一下午，翻那本日记。
从第一页翻到最后一张有字的页码。
九月的字迹圆圆的。十一月的字迹变了，用力很重。
他翻到那张纸条的位置。
"不是你的错。"
被泪痕或水渍洇过的横格纸。
他把纸条抽出来，对着窗外的光看。
墨迹晕开的地方，边缘有一点蓝色。
和他的指腹一样的蓝色。
他把纸条夹回去，合上日记。
等待的时间里，他把值班记录本翻开。
在"电话已打"下面，另起一段。
笔尖悬在半空。
他不知道写什么。
最后写了三个字：她听到了。
晚上八点，诊所下班。
封不觉没有走。
他坐在值班室里，等。
十点。走廊的灯自动暗了一半。
十一点。他去了一趟洗手间。
走廊的回声和白天不一样。每一步都传得更远。
十一点半。他回到值班室，把门关上。
没有锁。
他想了想。还是没有锁。
十二点差五分。
走廊尽头传来一声响。像旧木板被踩了一脚。
封不觉把手放在桌上。
指腹的蓝痕开始发热。不是疼。是热。
像有人从另一端握住了他的手指。
午夜零点。
值班室的温度降了一点。
空调还在运转，但吹出来的风和白天不一样。
封不觉抬头。
女孩站在诊室角落。和昨晚同一个位置。
校服蓝白配色，长袖，袖口有点脏。
头发扎着，碎发在耳朵前面。
她的嘴唇没有在动。只是看着他。
封不觉说："我告诉她了。"
他的声音在空气里是正常的。没有昨晚那种被掐掉的感觉。
女孩没有反应。
但她站的位置，空气有点扭。
像热气从地面升起来，但方向是反的。
封不觉从桌上拿起日记本。
翻到纸条那一页。
"我把这个也给她看了。"
他指着纸条。"她哭了。但她知道了。"
女孩看着他。眼睛是黑的，没有反光。
但她的头动了一下。很小的幅度，向下。
那个动作的意思是：知道了。
封不觉把日记合上。
"我不知道你是谁。"
"我不知道你为什么选我。"
"但如果你想让我继续做——"
他顿了一下。
"我会继续。"
女孩抬起右手。
和昨晚一样，用食指在空气中从左向右画了一道线。
但这次不是在她自己的手腕上。是在空中。
一道没有痕迹的线。
画完之后，她的手垂下来。
然后开始消散。
不是瞬间消失。是慢慢变薄。
从脚底开始。像一张被火烧的纸，边缘卷曲，变成灰。
然后向上蔓延。
脚踝。膝盖。裙摆。
一寸一寸地变淡。
封不觉站起来。
他向前走了一步。又一步。
他站在女孩面前，看着她消散。
"你的名字是什么？"他问。
女孩已经说不出话了。她的嘴唇在动。
没有声音传过来。
但封不觉看懂了口型。
三个字的口型。他没能完全辨认出来。
最后一个字像是"月"。或者"乐"。
消散到肩膀的时候，有声音了。
很轻。像是从很远的地方传过来的。
"谢谢你。"
是女孩的声音。不是成年人。是孩子的声音。
十五六岁。和日记里一样的年纪。
封不觉没有回答。
他看着她的脸最后变淡。
眼睛。鼻子。嘴唇。
然后什么都没有了。
诊室安静了。
空调嗡的一声重新响了。
饮水机咕咚冒泡。
这些声音回到正常长度。
封不觉低头。
左手心里有一个东西。
极小。像一粒盐。
白色的。不是透明的，是乳白色，像被光照透的云。
他把它捏起来。
指腹的蓝痕碰到它的时候，烫了一下。
不是灼伤的烫。是温热的。
像有人从另一端握着他的手。
他把结晶举到灯下看。
里面没有东西。就是一团淡白色的光。
或者不是光。是什么别的东西。
他把结晶放在桌上。
然后从抽屉里拿出值班记录本。
翻到新的一页。
笔尖在纸上停了很长时间。
然后写了一行字：
零点十五分。消散完成。遗留物：白色结晶一粒。来源：不明。
他看着这行字。
"完成"——他用了这个词。
不是"消失"。是"完成"。
他把笔放下。
左手腕上的青痕还在。
但颜色比早上更浅了。
像一支用淡了的铅笔画的线。
封不觉把结晶放进抽屉。
用一张纸巾包着，放在病历夹旁边。
然后他坐在转椅上，转了个方向，面对窗户。
窗外的路灯照着对面砖墙。
和昨晚一样。和每个晚上一样。
但有什么东西变了。
他摸了一下左手腕。
青痕的位置。皮肤下面没有跳动感了。
只是平平的。一道痕。
他闭上眼睛。
没有女孩站在天台边缘的画面。
没有坠落的记忆。
脑子里什么都没有。很空。
这是他两天来第一次感到"空"。
他保持这个姿势坐了很久。
直到走廊的灯在天亮前自动熄灭。
然后他开始写。
在值班记录本上，把昨晚没写完的内容补完。
他不知道这东西算不算是病历。
但如果他不写下来，以后就没有人会知道。
他写了女孩日记里的内容。写了纸条。写了电话。
写了消散的过程。写了那三个字。
"谢谢你。"
他写到这三个字的时候，笔尖顿了一下。
纸上洇了一个墨点。
他没有涂掉。让它留在那里。
六点。窗外的天开始变亮。
封不觉把记录本合上。
他从抽屉里拿出那颗白色结晶。
放在手心。很小。很轻。
没有重量，但存在。
他把结晶放进口袋。
然后站起来，走到门口。
门把手是凉的。和昨晚一样凉。
他拉开门。
走廊亮着灯。间距均匀。没有一盏灭的。
地面干净。没有湿痕。没有脚印。
他走回值班室，把门关好。
在椅子上坐下，等白天的人来。
口袋里的结晶有点发热。
隔着布料，他能感觉到那个温度。
像一颗不会冷却的小石头。
七点十五分，前台的女人来开门。
她从值班室门口路过，探头看了一眼。
"今天来这么早？"
"值夜班。"
"哦。"她点点头，走了。
封不觉从口袋里掏出结晶。
在早晨的光线下，它是半透明的。
里面有一点什么东西在动。
不是真的在动。是他眼花了。
他把结晶放回口袋。
然后翻开值班记录本的新一页。
在上面写：第七点三十分。第一例，完成。
他看着"第一例"三个字。
不知道为什么会用"例"这个字。
像医生写病历的措辞。
但他不是医生。只是实习生。
他把这个字划掉。
在旁边重写：第一件。
然后又划掉。
最后写：第一个。
他把笔放下。
口袋里的结晶在发热。
那个温度一直持续到他走出诊所大门。
三月的风吹在脸上，不冷。
他抬头看了一眼天。
太阳还没完全升起来，东边是淡红色的。
像火烧云。但不是云在烧。
是光在慢慢渗透进来。
封不觉走下台阶。
左手插在口袋里，手指握着那颗结晶。
他走过街角，等公交车。
车站有三个人。一个老太太，两个中学生。
他站在他们后面。
老太太回头看了他一眼。
她的眼睛在封不觉身上停了一秒。
然后转回去。
封不觉知道她在看什么。
不是看他的脸。是看他的肩膀后面。
他回头。
什么都没有。只有早晨的阳光照在站牌上。
但他感觉有什么东西在那里。
不是昨晚那个女孩。
是别的。更淡。更远。
像有人在排队。
公交车来了。
他上车，刷卡，走到最后一排坐下。
车窗外的城市在慢慢醒过来。
便利店开门，早餐摊支起来，穿校服的学生骑着车掠过。
他把结晶从口袋里掏出来，放在手心。
淡白色的。在阳光下几乎看不见。
但它确实在那里。
他合上手指。
指腹的蓝痕碰在结晶上。
有一瞬间的疼。像针扎。
然后变成热。
像有什么东西从结晶里流出来，渗进他的皮肤。
他把手摊开。
结晶还在。大小没变。颜色没变。
但他的指腹蓝痕深了一点点。
他看了很久。
然后把结晶放回口袋。
车到站的时候，阳光已经铺满了街道。
封不觉下车，走回出租屋。
楼梯间的灯坏了。他摸着扶手往上走。
在三楼拐角停了一下。
左手腕的青痕还在。但变得很浅了。
像随时会褪掉。
他上了四楼，打开门。
屋里和早晨离开时一样。
床上的被子没叠。日记本在床头柜上。
他走过去，拿起日记本。
翻开第一页。"今天开学。"
字迹圆圆的。蓝色圆珠笔。
他盯着这四个字看了很久。
然后合上本子，把它放进抽屉。
他躺在床上。
左手压在胸口。右手放在枕边。
口袋里的结晶隔着裤子布料贴着他的大腿。
温度还在。
他闭上眼睛。
没有女孩站在天台边缘。
没有坠落的记忆。
他看到一个空白。白色的。
然后睡着了。
这是三天来他第一次自然入睡。
没有噩梦。没有惊醒。
他睡了四个小时。
醒来的时候，下午的阳光从窗帘缝照进来。
一条光带落在地上。
他坐起来。
左手腕上的青痕只剩一个浅印子。
像被手指按了一下留下的白痕。
再过几天，大概会完全消失。
他摸了摸那个位置。
皮肤下面什么都没有了。平的。
他从抽屉里拿出值班记录本。
翻到最新的一页。
在上面写：三月某日。第一件执念，完成。遗留物：白色结晶。状态：已收容。
他不知道"收容"这个词对不对。
但他想不出来更好的词。
他把本子合上，放到一边。
然后从口袋里掏出结晶。
放在阳光下。
淡白色的。里面有东西在流转。
不是光。是记忆。或者不是记忆。
是别的什么。
他看了很久。然后把它握在手心。
温度正好。不烫。不凉。
像一个人的体温。
封不觉把结晶放在床头柜上。
和日记本放在一起。
墨绿色的封面。淡白色的结晶。
两个东西摆在一起，有一种奇怪的对称。
他躺回去，看着天花板。
三月的阳光在房间里慢慢移动。
从床头移到床尾。
他跟着那道光带移动视线。
什么都不想。
这是他能做到的最好的休息。
晚上他去诊所值第二个夜班。
走廊的灯亮着。候诊沙发上的绿萝叶子擦过扶手。
一切都和第一天一样。
但他知道有什么东西变了。
他坐在值班室里，把记录本翻开。
在"第一个"下面画了一条线。
然后另起一行。
笔尖悬在半空。
他不知道下一个会是什么。
但他知道会有下一个。
抽屉里的结晶在发热。
隔着木头，他能感觉到那个温度。
像有人在敲门。
不是用拳头。是用手指。
敲了三下。很轻。
他关上抽屉。
然后把椅子转过来，面对门口。
等。
