# 第0006章 第二件遗物

凌晨三点。
封不觉拉开门。
走廊空无一人。灯亮着。间距均匀。地面没有一点痕迹——没有湿痕，没有脚印，没有安全帽的碎片。
但指腹蓝痕还是热的。
刚才门外确实有人。
他在走廊里站了十秒。左翼。右翼。电梯口。全部是空的。
回到值班室，翻开记录本。在"敲门者"那行下面写了：三点后消失。未进入。走廊无异物。
合上本子。躺在折叠床上闭眼。
睡着之前脑子里有一个问题——明天晚上他还会来吗。
睡了四个小时。醒来是因为走廊灯自动亮了。
七点。他把几乎用尽的结晶从抽屉里拿出来。放在桌上。在光线下转了一下——只剩一粒微不可见的白点。比针尖还细。
第一颗结晶。用完了。
他把那粒白点捏在指尖上。
犹豫了一下。
然后放进一个空药瓶里。旋紧盖子。
在瓶身上写了"第一颗"三个字。
黑色马克笔。字很小。
然后把瓶子放进抽屉最里面。
白天值班只有几个病人。
没有带影的。没有异常的。一切正常。
封不觉注意到其中一个病人——中年女人，手上戴着一枚旧戒指。款式很老。至少二十年了。
她坐下的时候先用手指碰了碰椅背——一个无意识的动作。
不是确认椅子还在。是她丈夫生前总替她拉椅子。
她的影很淡，几乎看不见。但不是没有。
封不觉坐在值班室里，把值班记录本翻开。
从第一天到第五天的记录都在这儿——日记出现，女孩，电话，消散，结晶，刘老，灰影，规则，敲门。
他翻回第一页。在最早的那行字——"零点四十一分，日记一本，来源不明，无法确认为幻觉"——旁边，用红笔画了一个圈。
然后在下面写：第一件，已完成。
字很小。红笔。像一个归档章。
他把第一页到第四页折了一个角。表示这部分结束了。
前台女人中午路过值班室，探头看了一眼。
"主任让我问你——排班表要不要调？你连着值了好几个夜班了。"
"不用。"
她看了看他。
"你眼睛下面有黑眼圈。"
"睡不够。"
她没再说什么。走了。
傍晚六点。诊所空了。
封不觉打扫值班室。把旧饼干袋扔掉。杯子洗了两遍。桌面用湿布擦了一遍。饮水机的水换了新的一桶。做完之后他坐在转椅上。看着门。
不是午夜。但他知道今晚会有。昨晚已经敲过门了。今晚会再来。
十一点半。空调停了。
值班室的空气慢慢变闷。不是热——是停滞。像被什么东西从外面压着，往屋里挤。
走廊灯自动暗了一半。
封不觉把门关好——没有锁。他想了想。然后把锁扣按下去了。咔嗒一声。这是他第一次锁门。
他坐回椅子上。手放在桌面上。指腹朝上。蓝痕在台灯下颜色深了一个色号。
等。
十一点五十分。指腹蓝痕开始发热。
不是疼。是那种被什么东西从另一头握住了的感觉。温度缓慢升高——从温热到有点烫。
他低头看手。蓝痕的颜色比白天深了两个色号。食指内侧的白痕——三道之间开始模糊，边界在融合。不是三道了。是一条宽的。
他把手放在桌上。盯着门。锁扣还按着。
十二点整。
诊室钟锁在12。走廊钟锁在12。五号诊室的门把手温度开始下降——他没去测，但他知道。空气变了——比刚才更干，更薄。像海拔突然升高了的那个瞬间。
走廊传来脚步声。
不是软底鞋。是靴子。硬底。踩在地砖上每一步都有回音。闷的。沉的。像重物一下一下落在水泥地上。
封不觉站起来。
脚步声走到值班室门口。停了。
安静。
三秒。
然后敲门。三下。不重——但比昨晚沉。不是手指关节。是拳头。或者是什么硬的东西。比如安全帽的帽檐。
封不觉走过去。拨开锁扣。拉开门。
门口站着一个男人。
工装服。蓝色，洗得发白，左胸口袋撕掉了一半。裤子上有干涸的泥浆。头发上沾着水泥灰——灰白色的粉末，混在发茬里。
安全帽裂成两半，扣在头上。
裂缝从头盔顶部一直裂到帽檐。
帽檐下面——一张四十多岁的脸。
皮肤黝黑。
眼白是黄的。
嘴唇干裂，翕开一道缝。
喉咙里传出来的是工地噪音——电钻、钢筋碰撞、水泥搅拌机的轰鸣。
不是人的声音。
封不觉退了一步。让开路。
工人灵体走进来。
不是飘——是走。
每一步踩在地砖上都有闷响。
靴子底下粘着干掉的混凝土碎屑，踩在值班室地面上，一粒一粒地滚落。
安全帽上掉下来一块塑料碎片，在地上弹了一下，停在封不觉脚边。
他弯腰捡起来——塑料是冷的。不是冰的冷。是地底下的冷。
工人站在值班室中央。转身面对封不觉。嘴在动——嘴唇一开一合，发出断断续续的音节："脚手……不——不是掉……不是掉……"
他说不出完整的句子。
不是不想说。是执念太强而记忆太碎了——像一叠被撕散后胡乱钉回去的纸。每一页都在。但顺序全错了。
封不觉伸手碰到安全帽。指腹蓝痕接触帽檐的一瞬间——疼。不是针扎。是锤击。一股冲击从手指一路撞到手腕。他咬着牙没有缩手。
然后画面涌进来。
不是日记本那种"看电影"式的记忆。
是更暴力的——他站在那个工人站的位置。
头顶是脚手架。
太阳在西边。
光线从钢管之间漏下来。
手上有工具。
虎口是裂的——老茧。
不是封不觉的手。是工人的手。
粗糙。
指甲里嵌着机油。
有人在后面喊了一声什么。隔得远。没听清。
然后后背被推了。
不是手推的。
是什么硬东西——铁管。或者扳手。
冲击从后背撞进来，脊椎被顶了一下。
手在空气里乱抓——什么也抓不住。
脚手架的铁管在眼前旋转。
天空。楼层的窗户。天空。地面。
封不觉跪在地上。左手撑地。膝盖磕在值班室地面上，发麻——和第一天一样。
这次更重。不是被画面压下去的。是被撞下去的。
后背还在发麻——那个被推的位置。隔着时间。
安全帽在他手里。塑料边缘割进了手心。他没感觉到。
指腹蓝痕还在痛——持续的，一波一波。
食指皮肤上，三道白痕之间的空隙已经全部融掉了——变成一条模糊的白线。
从左到右，食指第一指节的位置。
像被细砂纸磨出来的那种白。
他把安全帽放在桌上。手在抖。
安全帽内衬里滑出来一样东西。
一张照片。边缘卷了。
照片上五六个人站成一排。
工人的笑脸——照片上的他还活着，戴着一顶完整的黄色安全帽。最右边。
倒数第二个人在笑。
其中一个的手搭在另一个人肩上。
背景是工地脚手架。
洗出来的照片背面有圆珠笔写的字——XX建筑公司，三年前的日期。
墨迹淡了。但能看清。
照片有折痕。被人反复打开又合上过——不是封不觉。是工人。死之前就经常看。
封不觉把照片凑近台灯，人脸一个一个看过去。
最左边的是个年轻人。二十出头。站姿有点歪——肩膀一边高一边低。
中间几个在笑。最后一个是工人自己——黄色安全帽，完整的。那时候还没裂。
他看完正面，翻到背面。然后抬头。工人灵体站在他面前。嘴还在动——这次音节能连起来了："推——推我的人——"
他抬起手。手指——工人的手指，指甲还是裂的——指着照片。"照片上。"
封不觉低头看合照。五个人。六个人。工人是其中之一。推他的人——也在这张照片里。
"哪一个是——"
工人张嘴。喉咙里只传出电钻声。尖锐的。持续的。不是他不想说。是死的时候声带被什么堵住了——不是恐惧。是水泥。喉咙里全是水泥。
封不觉翻开值班记录本。翻到新的一页。在第一行写了：第二件。
另起一行：工人。安全帽。工地合照。XX建筑公司。日期三年前。
然后他合上本子。看着工人。
工人站在那儿。
安全帽碎片还在往下掉。一片。两片。掉在地上弹一下。
然后塑料碰到地砖的声音停了——安全帽不再掉碎片了。
工人低头看了看自己的手掌。
掌心上是一道一道的疤——旧伤。不是死的时候留下的。是活着的时候。
他攥紧了拳头。不是恨。是别的——是他活着的时候的手，在做死后的动作。
封不觉把合照夹进记录本里。
然后把记录本放进抽屉。和第一颗结晶的空药瓶放在一起。
他抬头看工人。
工人站在诊室角落。和第一晚女孩站的位置一样。穿着工装。血没干——也不会干了。那是死的时候的样子。死后一直保留的。
封不觉把门关好。没有锁。然后坐回椅子上。
凌晨两点半。
工人站在角落里的姿势一直没有变。他在等。和女孩一样。会一直站到执念完成为止。
封不觉翻开记录本。在第一案的收尾页写了一个"完"。然后在新的一页开头写了：第二件——不是工伤。是谋杀。
他把本子合上。对着角落说了一句。"你认识推你的人。"
工人没有反应。
"我会查。"
工人还是没有反应。眼睛看着他身后的某个地方——不是在看封不觉。是看他背后。封不觉回头。只有墙。
再转回来的时候，工人眼睛里的某样东西动了。不是瞳孔——是更深的地方。像一盏灯在很深的水下闪了一下。
封不觉把手放在桌上。指腹的蓝痕还在发热。
白线已经完全成形——食指第一指节。一条。宽的。
不是青不是蓝不是白的颜色——是介于三者之间的某种浅灰。
像粉笔画在潮湿的纸上那种模糊的线。
他看了很久。然后把手指握进掌心。攥了一会。
他把手指摊开。白线在台灯光下几乎看不见——和指腹那条蓝色墨痕不一样。蓝色的在灯光下会反光。白的不会。白的吸收光线。
像一条缝。不是皮肤上的。是别的什么——更深的。
他把手指重新握进掌心。
攥了一会。
温度正常。
