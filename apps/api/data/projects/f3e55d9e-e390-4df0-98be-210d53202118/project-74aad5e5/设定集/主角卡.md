# 主角卡

## 姓名：呆英俊

## 性格

呆英俊表面看似木讷、不善言辞，实则内心坚毅如铁。他习惯隐忍，面对嘲讽和欺凌从不轻易动怒，而是默默积蓄力量。他的坚毅源于从小经历的磨难——出身卑微却怀揣梦想，即使被全世界否定，也从未放弃对强大的渴望。他重情重义，一旦认定朋友，便会拼死守护；但同时也固执己见，有时会因过度隐忍而错失良机。

## 能力

- **初始状态**：无斗气修为，仅靠肉身力量，但耐力惊人。
- **核心能力**：
  - 斗气觉醒后，主修火系斗气，擅长爆发性攻击。
  - 因灵魂力量强大，对炼丹有天赋。
  - 护道老爷爷传授的“帝级功法”残卷，可吞噬异火提升修为。
- **特殊天赋**：
  - 超强意志力：能在绝境中突破极限，越级战斗。
  - 敏锐直觉：对危险和机缘有本能感知。

## 背景故事

呆英俊出生在斗气大陆一个偏远小村的贫寒家庭，父母早逝，靠邻里接济长大。他天生斗气感应微弱，被检测为“废柴体质”，受尽冷眼。但他不甘平庸，十五岁时独自前往炎龙帝国，试图加入宗门改变命运，却因天赋太差被拒之门外。流浪期间，他意外救下一只受伤的魔兽，却因激怒当地贵族被打成重伤。濒死之际，他体内封印的“护道老爷爷”——一位上古斗帝的残魂——苏醒。老爷爷告诉他：他的废柴体质实则是“混沌灵体”，因天地异变被封印，需吞噬异火才能解封。从此，呆英俊踏上逆袭之路。

## 核心欲望与缺陷

- **核心欲望**：证明自己不是废柴，守护珍视之人，最终成为斗帝，揭开混沌灵体的秘密。
- **缺陷**：
  - 过度隐忍，有时会压抑情感，导致爆发时失控。
  - 因童年阴影，对强者有本能戒备，难以信任他人。
  - 修炼初期资源匮乏，常因冲动冒险而陷入危机。

## 成长路线

1. **起步阶段**：在护道老爷爷指导下，炼化第一种异火，觉醒斗气，突破斗者。
2. **成长阶段**：加入散修联盟，结识伙伴，参与秘境探险，吞噬更多异火，实力攀升至斗王。
3. **巅峰阶段**：对抗魔族入侵，揭露上古之战的真相，最终突破斗帝，重塑大陆秩序。

呆英俊的逆袭之路，将是他从被嘲笑到被仰望的蜕变，也是他找回自信与爱的旅程。
