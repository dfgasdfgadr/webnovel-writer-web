# 力量体系设定

## 修炼等级

斗气修炼分为九大境界，每境分初、中、高、巅峰四阶：
1. **斗者**：初入修炼，凝聚斗气，可施展基础斗技。
2. **斗师**：斗气外放，形成气旋，威力大增。
3. **大斗师**：斗气凝实，可化形攻击。
4. **斗灵**：觉醒灵魂力量，感知天地。
5. **斗王**：斗气化翼，凌空飞行。
6. **斗皇**：掌控领域，影响一方天地。
7. **斗宗**：开辟空间，瞬移穿梭。
8. **斗尊**：领悟法则，肉身不朽。
9. **斗帝**：传说之境，超越轮回。

## 体系分类

斗气按属性分为七大系：
- **火系**：攻击力强，爆发高。
- **冰系**：控制与防御，擅长持久战。
- **风系**：速度与敏捷，适合刺客。
- **雷系**：破坏力惊人，但难以掌控。
- **土系**：防御与辅助，稳定可靠。
- **木系**：治疗与生机，炼丹必备。
- **无属性**：兼容所有斗气，但修炼缓慢。

## 力量来源

斗气来源于天地间的灵气，修炼者通过功法吸收灵气，转化为自身斗气。此外，还有三种特殊力量：
- **血脉之力**：某些家族传承的先天斗气，觉醒后威力倍增。
- **丹药**：通过炼丹术淬炼灵药，可加速修炼或突破瓶颈。
- **功法与斗技**：高深功法决定修炼速度，斗技决定战斗方式。

## 关键规则

- **天赋限制**：修炼速度与天赋挂钩，但后天的机缘和努力可弥补。
- **境界压制**：高境界强者对低境界有绝对压制，但特殊斗技可破界。
- **天劫**：突破大境界时需渡劫，失败则魂飞魄散。
- **灵魂力量**：修炼到斗灵后，灵魂可离体探查，是炼丹和阵法的关键。
