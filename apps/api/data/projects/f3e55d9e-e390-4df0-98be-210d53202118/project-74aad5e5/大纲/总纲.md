# 斗破苍穹之逆命

**题材**：玄幻

**核心卖点**：废柴少年，逆天改命，以斗气破苍穹！

## 故事概述

在斗气大陆，斗气是衡量一切的标准。少年呆英俊，本是家族天才，却因一场意外斗气尽失，沦为受尽白眼的废柴。但他坚毅隐忍，从不言弃，在一次奇遇中获得了远古传承，重新踏上修炼之路。面对家族的冷落、同辈的嘲讽、强敌的追杀，他凭借不屈的意志和过人的智慧，一步步突破斗者、斗师、斗王，直至斗帝。在成长途中，他结识挚友，遭遇宿敌，揭开大陆隐藏的秘密，最终以斗气巅峰之力，守护所爱之人，成就一段逆天传奇。

## 分卷规划

- **第1卷 陨落与重生**：呆英俊斗气尽失，从天才沦为废柴，受尽屈辱。在家族试炼中意外觉醒远古传承，重新开始修炼，并结识第一位挚友。（100章）
- **第2卷 崛起之路**：呆英俊修为突飞猛进，在学院大比中一鸣惊人，引起各方势力注意。遭遇第一个宿敌，并开始探索斗气大陆的远古遗迹。（100章）
- **第3卷 风云际会**：大陆局势动荡，远古传承的秘密逐渐浮出水面。呆英俊卷入多方势力的争斗，在生死磨砺中突破斗王，并结识红颜知己。（100章）
- **第4卷 破茧成蝶**：呆英俊修为达到斗皇巅峰，与宿敌展开最终决战。揭开远古传承的真相，发现大陆危机的源头，为守护亲友而战。（100章）
- **第5卷 斗破苍穹**：呆英俊突破斗帝，联合大陆强者对抗终极黑暗势力。以逆天之力重塑斗气大陆秩序，成就一代传奇，与挚爱共度余生。（100章）
