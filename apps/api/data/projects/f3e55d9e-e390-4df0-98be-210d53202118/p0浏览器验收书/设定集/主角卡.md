# 主角卡

## 姓名

林玄

## 性格

林玄性格内敛而坚韧，表面冷漠寡言，实则内心炽热。他习惯用行动代替言语，面对困难时从不退缩，但偶尔会因过度执着而显得固执。他极度重视同伴，为了守护重要之人可以不顾一切。同时，他有一种天生的好奇心，对未知事物充满探索欲，这驱使他不断深入神陨遗迹。

## 能力

- **基础能力**：凝灵境巅峰（故事开始时），擅长元素流中的土元素掌控，能施展“岩盾”、“地刺”等初级法术。
- **独特天赋**：林玄的灵魂与一块神祇残骸碎片意外融合，使他能感知到灵源中的“记忆残留”——即神祇生前的片段记忆。这赋予他破解遗迹陷阱、预知危险的能力，但也让他的灵魂承受着被神祇意志侵蚀的风险。
- **战斗风格**：稳健防御型，利用土元素构筑护盾与地形优势，伺机反击。

## 背景故事

林玄出生在中央厚土平原的一个偏远村落，父母是普通的灵源矿工。十年前，神陨教团的余孽袭击了他的村庄，屠戮了所有村民，林玄在逃亡中坠入一处神陨遗迹，被一块碎片刺入胸口。他奇迹生还，但碎片融入了他的灵魂。此后，他被路过的厚土联邦散修“铁臂”赵洪收养，学习土元素修炼。赵洪在一年前被仇家所杀，林玄独自踏上修行之路，誓言寻找神陨教团复仇。

## 核心欲望与缺陷

- **核心欲望**：找到神陨教团的幕后主使，为父母和村民复仇；同时，解开神祇碎片中隐藏的秘密，或许能触及神陨境的真相。
- **缺陷**：林玄的复仇执念让他容易冲动，有时会忽视潜在的危险。此外，神祇碎片在提升他感知力的同时，也让他变得多疑，难以信任他人。他的灵魂与碎片融合并不完美，每当情绪剧烈波动时，碎片会试图侵蚀他的意识，导致短暂的失控。

## 成长路线

- **短期目标（炼气期）**：突破启灵境，掌握土元素的进阶技能“岩石铠甲”，并在神陨遗迹中寻找线索，追踪神陨教团。
- **中期目标（筑基期）**：在历练中结识同伴，组建自己的小队。学会如何控制神祇碎片的力量，避免被侵蚀。同时，发现神陨教团与裂隙之灾的关联，卷入大陆的暗流。
- **长期目标（金丹期及以后）**：逐步揭露神陨教团的阴谋，最终面对幕后黑手——一位试图利用神祇残骸重塑神格的灵皇境修士。在战斗中，林玄将面临选择：是彻底消灭神祇碎片，还是将其完全炼化，踏上成神之路？他的选择将决定玄元大陆的未来。
