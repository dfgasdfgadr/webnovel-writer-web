# 力量体系设定

## 修炼等级

修炼体系基于“灵源”的掌控程度，分为九阶：
1. **凝灵境**：初识灵源，将其引入体内，强化体质。寿命延长至150岁。
2. **启灵境**：灵源在体内形成核心，可释放基础元素技能。寿命300岁。
3. **通灵境**：灵源与灵魂融合，感知周围元素波动，能施展中阶法术。寿命500岁。
4. **化灵境**：灵源外放，形成护体灵甲或武器，可短暂飞行。寿命800岁。
5. **融灵境**：与某种元素深度共鸣，能操控大范围元素力量。寿命1200岁。
6. **灵王境**：灵源凝聚为“灵核”，可开辟小范围领域，压制低阶对手。寿命2000岁。
7. **灵皇境**：灵核进化为“灵丹”，领域扩大至十里，可引动天象。寿命5000岁。
8. **灵圣境**：灵丹化婴，半只脚踏入神域，能短暂扭曲现实法则。寿命万年。
9. **神陨境**：破碎灵婴，重铸神格，理论上可成神，但万年来无人达成。

## 体系分类

修炼者根据天赋与选择，分为三大流派：
- **元素流**：专精一种元素（金、木、水、火、土），追求极致的元素掌控力。例如火元素修士能以火焰焚天煮海，但容易被相克元素压制。
- **混沌流**：同时驾驭多种元素，但难以精通。混沌流修士通常依赖技巧与战术，而非纯粹的力量。
- **灵体流**：将灵源用于强化肉身，修炼成“灵体”，如金刚不坏之身或极速之躯。灵体流近战无敌，但远程能力薄弱。

## 力量来源

所有力量的根本是“灵源”，但获取方式多样：
- **自然吸收**：在灵源浓郁的区域（如神陨遗迹）修炼，效率最高。
- **灵晶**：由灵源凝结的晶体，可直接吸收，但产量稀少，是大陆上的硬通货。
- **吞噬**：猎杀妖兽或吸收他人灵核，但容易导致灵源污染，走火入魔。

## 关键规则

- **元素相克**：金克木、木克土、土克水、水克火、火克金。相克关系在战斗中影响巨大，可造成额外伤害。
- **灵源污染**：吸收被神祇残骸污染的灵源，会导致意识扭曲，甚至被神祇意志夺舍。
- **领域压制**：达到灵王境后，领域内的低阶修士会被削弱，高阶修士则能抵消部分压制。
- **神陨禁术**：传说中，神陨境修士可施展禁术，以生命为代价召唤神祇残影，但万年来无人成功。
