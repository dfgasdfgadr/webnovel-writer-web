# 世界观设定

## 世界背景

玄元大陆，一片由破碎神域碎片拼接而成的奇异世界。万年前，一场名为“神陨之战”的浩劫撕裂了天穹，无数神祇陨落，其残骸与意志散落于大地，形成了这片大陆。这个世界没有完整的法则，只有混乱与机遇并存。大陆被五大元素——金、木、水、火、土——所笼罩，每一种元素都对应着不同的地域特征：极北的冰原由水元素主宰，南方的熔岩荒漠由火元素统治，东方的苍翠森林是木元素的领地，西方的金属山脉由金元素塑造，而中央的厚土平原则是土元素的根基。

大陆上的生灵，无论是人类、妖兽还是异族，都依赖“灵源”生存。灵源是一种从神祇残骸中溢出的能量，它滋养着万物，也赋予了生灵修炼的可能性。然而，灵源并非均匀分布：越靠近神陨之战的遗迹，灵源越浓郁，但同时也伴随着不可预知的危险——残存的神祇意志可能扭曲现实，制造出诡异的空间裂缝或时间漩涡。

## 种族与势力格局

玄元大陆的智慧种族主要分为三类：
- **人族**：数量最多，分布广泛，擅长利用灵源修炼，建立了三大帝国：北方的冰霜帝国、南方的烈焰帝国、以及中立的厚土联邦。人族内部派系林立，宗门、世家、散修之间明争暗斗。
- **灵族**：由元素意识凝聚而成的生灵，天生与某种元素亲和。他们生活在元素核心区域，如东方的木灵族守护着生命古树，西方的金灵族锻造着神器。灵族性格高傲，视人族为低等存在，但偶尔会与强大的人族修士合作。
- **妖兽**：由普通野兽或灵植吸收灵源进化而成，拥有智慧与力量。妖兽盘踞在荒野与遗迹中，形成了以兽皇为首的部落联盟。妖兽与人族、灵族之间时有冲突，但在面对共同威胁时也会结盟。

势力格局呈现三足鼎立之势：人族帝国、灵族议会、妖兽部落彼此制衡。此外，还有神秘的“神陨教团”——一群崇拜神祇残骸的疯子，试图复活陨落的神灵，他们潜伏在阴影中，是大陆上最不稳定的因素。

## 历史大事

- **神陨之战（万年前）**：神界内战导致无数神祇陨落，碎片坠入凡间，形成玄元大陆。战后，第一批生灵从神祇残骸中诞生。
- **灵源觉醒（八千年前）**：大陆上的生灵首次发现灵源的可修炼性，开启了修炼纪元。
- **元素战争（五千年前）**：人族、灵族、妖兽因灵源争夺爆发大规模战争，最终签订《元素公约》，划定各自领土。
- **裂隙之灾（两千年前）**：一道巨大空间裂缝在中央厚土平原开启，涌出无数虚空生物。三族联军付出惨痛代价将其封印，裂缝至今仍是禁忌之地。
- **神陨教团崛起（五百年前）**：一群修士在遗迹中发现神祇记忆，创立神陨教团，试图颠覆现有秩序。

## 地理环境

玄元大陆的地貌因元素影响而极端分化：
- **极北冰原**：永冻之地，冰川覆盖，隐藏着水元素的源头“冰晶之眼”。
- **南方熔岩荒漠**：火山密布，火元素核心“熔岩之心”位于地底深处。
- **东方苍翠森林**：古老树木遮天蔽日，生命之树“木灵之冠”屹立于此。
- **西方金属山脉**：矿脉丰富，金灵族锻造的“天工城”悬浮于山峰之间。
- **中央厚土平原**：土地肥沃，人族帝国的心脏地带，但裂隙之灾的封印遗址依然散发着不祥的气息。

此外，大陆上散布着无数“神陨遗迹”——神祇残骸的坠落点，这些地方灵源浓郁，但也危险重重，是冒险者与亡命之徒的乐园。
