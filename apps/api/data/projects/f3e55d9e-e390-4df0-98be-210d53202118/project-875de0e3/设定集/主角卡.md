# 主角卡：林夜

## 基本信息
- **姓名**：林夜
- **年龄**：18岁
- **身份**：天玄帝国边境小城“青石城”林家的旁系子弟，因血脉稀薄，被族人视为“废柴”。
- **外貌**：黑发黑瞳，身材瘦削，面容清秀但总带着一丝倔强，常年穿着破旧的灰袍。

## 性格特征
- **坚韧不拔**：虽被嘲笑欺辱十多年，但从未放弃修炼，每日偷偷在树林中苦练基础拳法。
- **冷静沉稳**：面对困境能快速分析局面，不冲动行事，善于隐忍。
- **重情重义**：对朋友（如青梅竹马苏瑶）和恩人（如老仆林伯）极为忠诚，但被背叛后会变得冷酷。
- **自卑与骄傲并存**：表面因废柴身份自卑，内心却坚信自己终有一日能逆天改命，怀有强烈的自尊心。

## 能力体系
- **初始状态**：淬体境三层（最低阶），灵根为“废灵根”，吸收玄气速度极慢，无法施展任何玄技。
- **金手指：逆命玄珠**：
  - **吞噬**：可吸收他人玄气（需接触），但会积累业力，业力过重会引发心魔。
  - **神通觉醒**：
    - 淬体境：洞察之眼（看穿对手玄气流动和弱点）。
    - 凝气境：玄气化刃（将吞噬的玄气转化为攻击）。
    - 化元境：时间回溯（短暂倒流三秒，每天一次）。
- **成长路线**：
  1. **初期**：利用洞察之眼在实战中偷学他人招式，通过吞噬妖兽和敌人玄气快速突破，但需小心业力反噬。
  2. **中期**：炼化法则碎片（如“速度法则”），结合逆命玄珠开发专属功法《逆命诀》，可吞噬玄技并改良。
  3. **后期**：对抗魔潮时，吞噬魔族魔气，领悟“阴阳平衡”，将业力转化为力量，最终对抗神魔残魂。

## 背景故事
林夜出身于林家旁系，母亲难产而亡，父亲林啸曾是林家天才，但因探索秘境失踪。林夜自幼被主家欺凌，住在破旧偏院，靠老仆林伯接济。16岁时，他在后山悬崖下发现一枚黑色珠子（逆命玄珠），珠子认主后，他发现自己能吸收他人玄气。但第一次使用后，他昏迷三天，醒来后珠子上多了一道血痕（业力标记）。从此，他一边隐藏能力，一边暗中修炼。18岁时，主家少爷林峰为抢夺他母亲遗物玉佩，当众羞辱他，林夜忍无可忍，在决斗中利用吞噬之力反杀林峰，暴露了逆命玄珠，被迫逃离青石城，踏上逆天之路。

## 核心欲望与缺陷
- **核心欲望**：
  1. 找到失踪的父亲，解开林家血脉之谜。
  2. 证明自己不是废柴，让所有嘲笑他的人跪伏。
  3. 最终目标：打破血脉歧视的阶级固化，建立一个人人平等的修炼世界。
- **核心缺陷**：
  1. **业力枷锁**：每次吞噬他人玄气，业力会侵蚀心智，若失控会堕入魔道。
  2. **情感软肋**：对苏瑶（青梅竹马，后被主家收买背叛）和林伯（被主家杀害）的执念，让他多次陷入陷阱。
  3. **过度依赖金手指**：初期过于依赖吞噬之力，导致根基不稳，后期需重新淬炼肉身。

## 成长路线概览
- **第一卷：青石城之变**（淬体→凝气）：逃离追杀，结识散修老乞丐，学会基础功法，首次使用吞噬业力险些走火入魔。
- **第二卷：万妖岭试炼**（凝气→化元）：进入秘境，炼化“速度法则碎片”，觉醒时间回溯神通，与妖族公主结盟。
- **第三卷：天玄城风云**（化元→金丹）：参加天玄大比，揭露林家血脉秘密（实为远古神魔后裔），被天玄帝国追杀。
- **第四卷：魔潮之战**（金丹→元婴）：对抗魔族，吞噬魔帝分身，领悟业力转化之法，成为大陆英雄。
- **终卷：逆命成神**（元婴→神境）：挑战神魔残魂，打破血脉枷锁，重塑天地法则，实现人人平等。
