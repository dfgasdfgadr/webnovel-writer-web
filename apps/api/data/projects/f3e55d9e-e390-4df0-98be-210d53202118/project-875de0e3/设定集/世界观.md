# 世界观设定：苍玄大陆

## 世界背景
苍玄大陆，一片被古老灵气笼罩的广袤世界，万族共存，强者为尊。天地间弥漫着名为“玄气”的能量，修炼者通过汲取玄气淬炼己身，突破极限，追求成神之路。然而，这片大陆并非永恒祥和——远古时期，一场“天崩之战”撕裂了虚空，导致神魔陨落，无数法则碎片散落人间，形成了如今混乱而危险的格局。玄气分布不均，东部苍龙域灵气浓郁，是修炼圣地；西部血渊域则被魔气侵蚀，沦为禁地；南部万妖岭妖兽横行，北部冰原雪域藏有上古遗迹。中央的天玄城，则是万族共尊的权力中心，掌控着大陆的秩序。

## 种族与势力
- **人族**：大陆主宰，分三大势力：天玄帝国（统治中央）、九霄剑宗（剑修圣地）、万宝商会（掌控资源）。人族内部等级森严，血脉决定天赋，平民子弟难有出头之日。
- **妖族**：栖息于万妖岭，以九尾天狐、龙族、凤族为首，天生肉身强横，但修炼缓慢。近年来因灵气复苏，妖族蠢蠢欲动，欲夺回失地。
- **魔族**：被封印在血渊域，以魔帝为首，擅长吞噬玄气，污染生灵。每千年一次的“魔潮”是大陆最大的威胁，需各大势力联手抵御。
- **古族**：隐世不出的神秘种族，如星辰族、时光族，掌握远古秘术，极少干预世事。

## 历史大事
1. **创世纪元**：天地初开，神魔诞生，玄气充盈，万物和谐。
2. **天崩之战（三万年前）**：神魔因争夺“太初本源”爆发大战，虚空崩塌，神魔陨落，法则碎片散落，大陆分裂为五域。
3. **人族崛起（两万年前）**：首位人皇“苍玄”统一五域，建立天玄帝国，确立玄气修炼体系，开创盛世。
4. **魔潮之灾（五千年前）**：魔族破封，席卷大陆，人族与妖族联手，以“九转封魔阵”将其封印，但代价惨重，无数强者陨落。
5. **灵气复苏（百年内）**：大陆玄气浓度突然暴增，远古遗迹纷纷现世，天才辈出，新一轮的争霸拉开序幕。

## 地理环境
- **苍龙域（东）**：灵气最浓郁，山峦叠嶂，瀑布飞流，有“苍龙山脉”横贯东西，内藏无数洞天福地。
- **血渊域（西）**：魔气弥漫，大地龟裂，寸草不生，唯有魔族遗迹和吞噬生灵的深渊裂缝。
- **万妖岭（南）**：原始森林覆盖，妖兽横行，中央有“万妖圣殿”，是妖族圣地。
- **冰原雪域（北）**：极寒之地，冰川覆盖，下方埋藏上古遗迹，但温度极低，非强者不可入。
- **天玄域（中）**：大陆心脏，天玄城坐落于此，城墙高耸，阵法密布，是万族贸易和权力博弈的中心。

## 社会规则
- **实力至上**：修炼等级决定地位，低阶修士只能为奴为仆，高阶强者可封侯拜相。
- **血脉歧视**：平民血脉天生修炼缓慢，贵族血脉则可继承先祖天赋，导致阶级固化。
- **弱肉强食**：野外秘境和遗迹中，杀人夺宝是常态，只有强者才能生存。
