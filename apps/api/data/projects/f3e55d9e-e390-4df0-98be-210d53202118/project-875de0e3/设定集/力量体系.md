# 力量体系：玄气修炼与万法之道

## 力量来源：玄气
玄气是天地间的基本能量，分为五种属性：金（锐利）、木（生机）、水（柔韧）、火（狂暴）、土（厚重）。修炼者通过功法吸收玄气，淬炼经脉，凝聚“玄丹”，从而获得超凡之力。

## 修炼等级（共九境）
1. **淬体境**：锤炼肉身，打通经脉，可容纳玄气。需引气入体，过程痛苦，资质差者终生无望。
2. **凝气境**：玄气凝聚成丹田气旋，能施展基础玄技，如“火球术”。
3. **化元境**：气旋化为“玄元”，玄气质量提升，可御物飞行，寿命延至两百年。
4. **金丹境**：玄元凝结成金丹，可引动天地之力，施展“领域”雏形。
5. **元婴境**：金丹破碎，诞生元婴，可灵魂出窍，初步掌握法则。
6. **化神境**：元婴与天地共鸣，可引动小范围天象，寿命千年。
7. **合道境**：与一条天地法则融合，成为“法则之主”，一念山河变色。
8. **破虚境**：超脱法则束缚，可撕裂虚空，遨游星海。
9. **神境**：掌控本源，永恒不灭，传说中已无人达到。

## 体系分类
- **体修**：主修肉身，以玄气淬炼筋骨，追求极致力量，如“金刚体”、“不灭金身”。
- **术修**：主修玄技，远程攻击，擅长元素操控，如“火系炎龙诀”、“冰封千里”。
- **阵修**：研究阵法，以玄气刻画符文，可困敌、杀敌、辅助，如“九转封魔阵”。
- **器修**：炼制法宝，以玄气温养，可提升战力，如“飞剑”、“护甲”。
- **毒修**：另类流派，以毒入道，诡异莫测，但易反噬。

## 关键规则
1. **天赋决定上限**：血脉和体质决定修炼速度，如“天灵根”可一日千里，“废灵根”则寸步难行。
2. **突破需契机**：每个大境界突破都需要“破境丹”或顿悟，否则可能走火入魔。
3. **法则碎片**：远古神魔陨落后，法则碎片散落天地，炼化后可获得特殊能力，如“时间加速”、“空间跳跃”。
4. **反噬与代价**：越级使用高阶功法或强行突破，会损伤根基，甚至爆体而亡。
5. **金手指：逆命玄珠**：主角意外获得的远古至宝，可吸收他人玄气转化为己用（但需承受因果业力），并每突破一境觉醒一项神通（如“洞察之眼”看破弱点、“时间回溯”短暂倒流）。

## 修炼资源
- **灵石**：蕴含玄气的晶石，分下、中、上、极品，用于修炼和布阵。
- **丹药**：炼药师炼制，如“聚气丹”、“破境丹”，但丹毒积累会污染经脉。
- **秘境**：远古遗迹，内藏功法、法宝，但危机重重。
