# 逆天废柴

**题材**：玄幻

**核心卖点**：废柴少年觉醒逆天血脉，一路碾压诸天万界，成就无上主宰。

## 故事概述

主角林尘本是青云城林家公认的废柴，灵根破碎，无法修炼，受尽欺凌。一次意外中，他激活了体内沉睡的太古神脉，获得逆天修炼天赋。从此，他踏上逆袭之路，以废柴之躯破而后立，横扫各路天才，夺宝炼丹，斩妖除魔。在成长过程中，他逐渐揭开上古秘辛，发现自己的身世与万界浩劫息息相关。面对即将到来的诸神黄昏，林尘以凡人之躯逆天而行，最终证道成神，守护苍生。

## 分卷规划

- **第1卷 初露锋芒**：林尘在青云城觉醒太古神脉，从废柴逆袭为天才，击败家族宿敌，踏上修炼征途。（100章）
- **第2卷 风云际会**：林尘进入玄天宗，在宗门大比中崭露头角，结识挚友与红颜，同时遭遇暗黑势力追杀。（100章）
- **第3卷 万界争锋**：林尘走出宗门，参与万界天才战，夺宝秘境，突破境界，逐渐揭开上古神脉之谜。（100章）
- **第4卷 逆天证道**：林尘对抗诸神势力，拯救万界于浩劫，最终以凡人之躯逆天证道，成就主宰。（100章）
